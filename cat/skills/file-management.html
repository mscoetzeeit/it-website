File & Data Management All Grades

Managing files is a core CAT skill examined in Paper 1 and needed for the PAT. You must be able to organise, find, rename, copy, compress and protect files in Windows File Explorer — and know your file types.

Files and Folders

File
A single collection of stored data with a name and an extension (e.g. report.docx)
Folder (directory)
A container that holds files and other (sub)folders, used to organise your work
File Explorer
The Windows program used to browse, open and manage files and folders (⊞ Win+E)
Subfolder
A folder inside another folder — creates a tree-like structure

File-naming conventions

  • Use short, meaningful names that describe the content (Term2_Budget.xlsx, not doc1.xlsx).
  • In the exam, save with the exact name given in the question, in the exact folder given.
  • Avoid these characters in file names: \ / : * ? " < > |
  • Don't change the extension when renaming — it tells Windows which program to use.

EXAM TIP — image formats

Know the difference: JPG = good for photos, smaller file (lossy). PNG = supports transparency, good for logos/graphics. GIF = supports simple animation, few colours. BMP = uncompressed, very large.

Common File Operations

Create a new folder
  1. Open the location where you want it (e.g. Documents) in File Explorer.
  2. Click New→Folder (or right-click → New → Folder, or press Ctrl+Shift+N).
  3. Type the folder name and press Enter.

COPY vs MOVE — the classic exam distinction

Copy makes a duplicate — the file exists in both places. Move (cut & paste) relocates it — the file exists in only the new place. Tip: drag within the same drive = move; drag to a different drive = copy.

Recycle Bin

Deleted files go to the Recycle Bin first, so you can Restore them. They are only gone for good once the Bin is emptied (or you used Shift+Delete). Deleting from a flash drive does not use the Recycle Bin — it's permanent.

A path describes exactly where a file is stored, from the drive down to the file:

Compressing & Extracting (Zip)

Compression reduces a file's size and can bundle many files into one .zip archive.

Why compress?
Takes up less storage space
Faster to email / upload (smaller)
Combine many files into one for easy sending
Some email systems only allow zipped attachments
Zip and unzip in Windows
  1. To compress: select the files/folder → right-click → Compress to ZIP file (or Send to → Compressed (zipped) folder). A new .zip appears.
  2. To extract: right-click the .zip → Extract All… → choose a location → Extract.

File Attributes

Attributes are settings that change how a file behaves. Set them via right-click → Properties.

Read-only
The file can be opened and read but not changed/overwritten — protects important files from accidental edits
Hidden
The file is not shown in File Explorer by default (turn on "Hidden items" in the View menu to see it)

Backups

WHY BACK UP?

A backup is a second copy of your data kept somewhere safe. If your device is lost, stolen, damaged, or hit by ransomware/a virus, the backup lets you recover your work. Losing your only copy of a PAT or project is a disaster you can avoid.

Backup methodNotes
External hard drive / SSDLarge capacity; keep it in a different place from the computer
Flash drivePortable; good for a single project; easy to lose
Cloud (OneDrive, Google Drive)Off-site & automatic; access anywhere; needs internet
The 3-2-1 rule

Keep 3 copies of important data, on 2 different types of media, with 1 copy off-site (e.g. the cloud). Back up regularly — a six-month-old backup won't save today's work.

EXAM TIP — practical file tasks

Paper 1 often starts by telling you to find, rename or move files in the exam folder, and ends by saving with a specific name. Read those instructions carefully — marks are awarded for the correct file in the correct place with the correct name, even before you open it.
