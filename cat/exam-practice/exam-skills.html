Exam Technique & Question Verbs All Grades

More CAT marks are lost to poor exam technique than to not knowing the work. If you learn what each command word wants and how to read mark allocations, you can add 10–15% to your mark without learning a single new fact.

1. The command word tells you HOW to answer

Every theory question starts with a verb (a "command word"). It tells you exactly what kind of answer earns the marks. Match the verb — don't just write everything you know.

Command wordWhat it wantsExample
Name / List / State / GiveA short fact or word. No explanation needed. Just give as many separate items as the marks."Name TWO input devices." → Keyboard, mouse.
DefineA precise meaning of the term."Define RAM." → Temporary, volatile memory holding data/programs currently in use.
Explain / DescribeSay what it is AND how/why it works. Full sentences."Explain how a firewall protects a network."
Distinguish / DifferentiateGive a point of difference for both things, ideally in the same sentence ("…whereas…")."Distinguish between RAM and ROM."
Motivate / Justify / Give a reasonGive a reason for an answer — link it to the scenario."Motivate why an SSD suits a laptop."
DiscussGive more than one side — advantages AND disadvantages / for and against."Discuss the impact of social media."
Suggest / RecommendGive an appropriate answer for the scenario and (often) say why."Suggest hardware for a graphic designer."
TabulatePut the answer in a table with the columns asked for."Tabulate two differences…"
The most common mistake

If the question says "Explain" but you only name the answer, you get half the marks at most. "Explain" needs the because/so that/which means part. Always finish the sentence.

2. The marks tell you HOW MUCH to write

The number in brackets after a question is your instruction for how many points to make.

(1)
One fact / one word. Don't waste time writing a paragraph.
(2)
Two separate facts — OR one point + its explanation, depending on the verb.
(2 × 2)
TWO things, each explained for TWO marks (e.g. "Give TWO advantages and explain each").
"TWO reasons"
You must give two different reasons. Repeating the same idea in other words earns one mark only.
EXAM TIP — count your points

Before moving on, count: does my answer have as many distinct points as there are marks? A 3-mark answer needs three different ideas. If you can only think of two, you know there's a mark to chase.

3. Scenario questions — use the scenario

Most CAT theory marks come from a scenario (a small business, a school, a sports club…). The examiner wants you to apply facts to that situation, not just recite a textbook list.

Weak (loses marks): "An SSD is fast and has no moving parts."

Strong (full marks): "An SSD suits the travelling sales rep because it has no moving parts, so it won't be damaged when the laptop is bumped in the car, and it boots faster so she can start quickly between meetings."

4. Don't repeat the question / don't be vague

  • "It is good / nice / better / faster" with no reason = no mark. Say why it is better.
  • Avoid brand names as answers unless asked (write "word processor", not only "Microsoft Word", in theory).
  • Never just rewrite the question. "Why is a backup important? — Because backups are important" earns nothing.
  • One device/term per mark — don't list five hoping one is right; examiners may mark the first answer only.

5. Paper 1 (practical) technique

Smart habits in the practical exam
  1. Save immediately and often (Ctrl+S). Save in the exam folder with the exact file name given.
  2. Read the whole question before doing anything — later parts sometimes change earlier steps.
  3. Do every sub-question, even if unsure. A wrong attempt can still earn method marks; a blank earns zero.
  4. Don't delete data unless told to. Markers open your file and check the result is there.
  5. Use Find & Replace and formulas rather than retyping — faster and fewer errors.
  6. Check formulas show a result, not an error like #REF! or #NAME?.
  7. Save once more at the end of every question. An unsaved file = a zero.

6. Time management

Roughly one mark ≈ one minute. If a question is worth 4 marks, don't spend 15 minutes on it. Leave hard parts, move on, and come back — there are easy marks waiting later that you don't want to miss because you ran out of time.

Final checklist before you hand in

Theory: every question attempted? Right number of points per mark? Verbs answered (explained, not just named)?
Practical: every file saved with the correct name in the correct folder? No formula errors? Spell-check run on Word documents?
