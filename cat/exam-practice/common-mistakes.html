Common Exam Mistakes All Grades

The same wrong answers cost learners marks every year. Each card shows a common mistake (✘) and the correct answer (✔). Skim these before a test — fixing a misconception is worth more than learning something new.

How to use this page

Cover the ✔ line and read only the ✘. Can you correct it in your own words? Then check. The ones you keep getting wrong are exactly what to revise on the topic pages.

Hardware

RAM stores files permanently.
RAM stores data temporarily while programs are running — it is volatile, so its contents are lost when the power goes off.
ROM and RAM are basically the same.
ROM holds permanent start-up (boot) instructions and is read-only; RAM is temporary working memory you can read from and write to.
A bigger hard drive makes the computer faster.
More storage gives more space, not more speed. To go faster, add RAM or switch to an SSD.
An SSD and an HDD store data in the same way.
An SSD uses flash memory (no moving parts, faster, more durable); an HDD uses spinning magnetic disks (slower, cheaper per GB).
A scanner is an output device.
A scanner is an input device — it captures (inputs) an image or text into the computer. A printer is the output device.
CPU speed is measured in gigabytes (GB).
CPU speed is measured in GHz. GB measures storage/memory capacity, not speed.

Software

Windows is application software.
Windows is system software — an operating system. Application software (Word, a browser) performs tasks for the user.
Freeware means the same as open-source.
Freeware is free to use but the source code is hidden. Open-source shares the source code so it can be viewed and changed.
Software updates only add new features.
Updates mainly fix bugs and patch security vulnerabilities; new features are a bonus.
Defragmenting the disk frees up storage space.
Defragmenting reorganises files so they open faster — it does not free space. Disk Cleanup frees space.
The operating system and antivirus are the same thing.
The OS runs and manages the computer; antivirus is a utility that detects and removes malware.

Networks

The internet and the World Wide Web are the same thing.
The internet is the global network (the infrastructure); the WWW is a service — the websites — that runs on the internet.
A modem and a router are the same device.
A modem connects you to the ISP/internet; a router directs traffic between your devices and networks (home units often combine both).
Wi-Fi is always faster than a cable.
A wired (UTP/fibre) connection is usually faster and more stable than Wi-Fi.
A LAN can cover a whole country.
A LAN covers one building/site; a WAN covers a large area. The internet is the largest WAN.
Bandwidth and a data cap mean the same thing.
Bandwidth = speed (how fast data moves); a cap = the amount of data you may use per month.

Internet & E-Communication

Incognito / private browsing makes you anonymous online.
It only stops your own device saving history and cookies. Your ISP, network admin and the websites can still see what you do.
Cc and Bcc do the same thing.
Cc recipients are visible to everyone; Bcc recipients are hidden from all other recipients.
Downloading and streaming are the same.
Downloading saves a complete copy to your device; streaming plays media in real time without saving it.
HTTP and HTTPS are equally safe.
HTTPS encrypts the data between you and the site (look for the padlock); HTTP does not.
A web browser and a search engine are the same.
A browser (Chrome, Edge) displays web pages; a search engine (Google) finds pages by keyword.

Social Implications

A virus and a worm are the same.
A virus attaches to a file and needs a user to run it; a worm spreads by itself across networks without a host file.
Phishing is a type of virus.
Phishing is social engineering — tricking a person into giving information. It is not malware.
Deleting a file removes it permanently.
It goes to the Recycle Bin first and can be restored — only gone once the Bin is emptied. Deleting from a flash drive is permanent.
Green computing just means recycling paper.
It also means saving energy (sleep mode, Energy Star devices) and safely disposing of e-waste.
A password you remember easily, like your name, is strong.
A strong password mixes upper and lower case, numbers and symbols, is 8+ characters and is not personal info.

Information Management & PAT

Data and information are the same thing.
Data is raw, unprocessed facts; information is processed data that has meaning and supports a decision.
Copy-pasting from a website is fine as long as you list it in the bibliography.
You must write in your own words (or use quotation marks). Copying text is plagiarism even when the source is listed.
Any website is a reliable source.
Evaluate authority, accuracy, currency and bias, and cross-check the fact in at least two reputable sources.

Practical — Excel, Access & HTML

=SUM(B2,B10) adds everything from B2 to B10.
A comma lists only those two cells. For a range use a colon: =SUM(B2:B10).
A formula can start without an = sign.
Every formula/function must start with =, otherwise Excel treats it as plain text.
VLOOKUP can search any column you point it at.
VLOOKUP looks up a value in the first column of the table and returns a value from a column to its right.
cellpadding and cellspacing are the same.
cellpadding = space inside a cell (around the content); cellspacing = space between cells.
A primary key may contain duplicate values.
A primary key must be unique for every record — no duplicates and no blank (null) values.
Absolute and relative references behave the same when copied.
A relative reference (B2) changes when copied; an absolute reference ($B$2) stays fixed.
Turn a mistake into a mark

In the exam, if a statement "sounds almost right", slow down — that's exactly where these traps live. Match the command word, give the right number of facts, and apply it to the scenario.
