Hardware & Processing

Q15

An advert reads: "Intel i5 2.8 GHz · 8 GB RAM · 512 GB SSD · 15.6″ Full HD".

1.1 Which item refers to the CPU and what does GHz indicate? (2)
1.2 Which item is the temporary memory, and what happens to its contents when the PC is switched off? (2)
1.3 Would adding more RAM speed up the computer? Give a reason. (1)

Show memo
  • 1.1 Intel i5 2.8 GHz ✓; GHz = the speed/clock rate — how many instructions it can process per second ✓
  • 1.2 8 GB RAM ✓; contents are lost / erased (it is volatile) ✓
  • 1.3 Yes — more RAM lets more programs/data stay in memory at once, so less slow swapping to disk ✓
Q23

2.1 What is a UPS and why is it useful? (2)
2.2 Distinguish between primary and secondary storage. (1)

Show memo
  • 2.1 Uninterruptible Power Supply — a battery backup ✓ that keeps the computer running briefly during a power failure so you can save work / shut down safely ✓
  • 2.2 Primary storage is internal memory (RAM/ROM) used while working; secondary storage is permanent storage (HDD/SSD/flash) for keeping data ✓

Social Implications & Security

Q85

8.1 Distinguish between a virus and a worm. (2)
8.2 What is phishing, and give ONE way to recognise a phishing email. (2)
8.3 Name ONE software measure to protect against malware. (1)

Show memo
  • 8.1 A virus attaches to a file/program and needs the user to run it to spread ✓; a worm spreads by itself across networks without a host file ✓
  • 8.2 A fake message pretending to be from a trusted source to trick you into giving personal info/passwords ✓; sign: suspicious links/addresses, urgent threats, spelling errors, asks for passwords/banking details ✓
  • 8.3 Antivirus software (also accepted: firewall, keep software updated) ✓
Q94

9.1 What is social engineering? (1)
9.2 Give THREE features of a strong password. (3)

Show memo
  • 9.1 Manipulating/tricking people into giving out confidential information or access (rather than hacking the system technically) ✓
  • 9.2 Any three: at least 8+ characters, mix of UPPER & lower case, includes numbers, includes symbols, not a real word/personal info, unique per site ✓✓✓

Practical — Excel, Access & HTML

P1 · Excel6

Sales values are in B2:B13 and the region for each is in A2:A13. The VAT rate (15%) is in cell E1.

P1.1 Count how many sales are greater than 1000.
P1.2 Add up only the sales for region "North".
P1.3 In C2, add VAT to the value in B2 — written so it can be copied down.
P1.4 Round the average of B2:B13 to 2 decimals.

Show memo
  • P1.1 =COUNTIF(B2:B13,">1000") ✓
  • P1.2 =SUMIF(A2:A13,"North",B2:B13) ✓
  • P1.3 =B2*(1+$E$1)  — note the absolute reference $E$1 ✓✓
  • P1.4 =ROUND(AVERAGE(B2:B13),2) ✓✓

Scenario & Interpretation Practice

Grade 11 exams often give you an advert to interpret or a small organisation to advise. Try these.

S1 · Interpret the advert4

A laptop advert reads: Ryzen 5 · 3.2 GHz · 16 GB RAM · 1 TB HDD + 256 GB SSD · Wi-Fi 6.

S1.1 Give ONE benefit of having both an SSD and an HDD in the same laptop. (1)
S1.2 Why is 16 GB RAM better than 8 GB for a user who edits photos? (1)
S1.3 What does "Wi-Fi 6" tell you about the laptop? (1)
S1.4 The advert does not list a GPU. What is the purpose of a (dedicated) GPU? (1)

Show memo
  • S1.1 The SSD gives fast boot & program loading; the HDD gives cheap, large storage for files ✓
  • S1.2 More RAM keeps large images in memory at once, so the software runs smoothly without lag/slowdown ✓
  • S1.3 It can connect wirelessly (to a WLAN) using the latest, faster Wi-Fi standard ✓
  • S1.4 A GPU handles/​renders graphics, needed for photo/video editing, 3D and gaming ✓
S2 · Mini scenario — the running club6

A community running club stores member records in an Access database and uses a small network in its office.

S2.1 Which field is the better primary key: MemberName or MemberID? Give a reason. (2)
S2.2 Name the device that lets the office computers share one internet connection and directs traffic. (1)
S2.3 The club posts race photos online. Give ONE privacy risk of sharing geotagged photos. (1)
S2.4 Give TWO features of a strong password for the club's admin login. (2)

Show memo
  • S2.1 MemberID ✓ — it is unique for every member, whereas two members could share a name ✓
  • S2.2 A router ✓
  • S2.3 Geotags reveal the exact location a photo was taken (e.g. a member's home), which stalkers or burglars could exploit ✓
  • S2.4 Any two: 8+ characters, mix of UPPER/lower case, includes numbers, includes symbols, not personal info ✓✓
