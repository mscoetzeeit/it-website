Section A — Objective questions

A1 · Multiple choice5

1.1 Which function looks up a value in the first column of a table and returns a value in the same row?  A) IF  B) VLOOKUP  C) COUNTIF  D) SUMIF

1.2 Deliberately overloading a server with traffic so it cannot respond is a …  A) phishing attack  B) DDoS attack  C) trojan  D) spam

1.3 An ISP slowing down certain traffic (e.g. streaming) is called …  A) shaping/throttling  B) caching  C) buffering  D) capping

1.4 Which file attribute prevents a file from being changed?  A) Hidden  B) Archive  C) Read-only  D) System

1.5 Combining text from two cells into one is done with …  A) LEFT  B) LEN  C) CONCAT/&  D) TRIM

Show memo
  • 1.1 B) VLOOKUP ✓
  • 1.2 B) DDoS attack ✓
  • 1.3 A) shaping/throttling ✓
  • 1.4 C) Read-only ✓
  • 1.5 C) CONCAT / & ✓
A2 · True / False (correct if false)3

2.1 A WAN connects computers within a single building.
2.2 Caching stores recently used web data so pages load faster next time.
2.3 A VPN makes your connection less secure.

Show memo
  • 2.1 FALSE — a LAN covers one building; a WAN covers a large geographic area ✓
  • 2.2 TRUE ✓
  • 2.3 FALSE — a VPN makes a connection more secure/private (it encrypts traffic) ✓

Hardware & Buying Decisions

Q16

A graphic designer who travels to clients needs a new laptop for photo & video editing.

1.1 Recommend a suitable specification for the CPU, RAM and storage, motivating each choice in terms of the designer's needs. (3 × 2)

Show memo

Each item: 1 for an appropriate spec + 1 for a relevant motivation linked to the scenario.

  • CPU: a fast multi-core processor (e.g. i7/Ryzen 7) ✓ — editing software is processor-intensive / renders faster ✓
  • RAM: 16 GB or more ✓ — large images/video need lots of memory to run smoothly without lag ✓
  • Storage: a large SSD (e.g. 1 TB) ✓ — fast access for big files + no moving parts, so safe while travelling ✓
  • (Also creditworthy: dedicated GPU, high-resolution display.)
Q24

2.1 Distinguish between a laser and an inkjet printer, mentioning ONE situation each is best suited to. (4)

Show memo
  • Laser uses toner powder, is fast and cheaper per page for bulk text ✓ — best for a busy office printing many text documents ✓
  • Inkjet sprays liquid ink, gives good colour/photo quality but is slower & ink is costly ✓ — best for a home user printing occasional photos ✓

Software & File Management

4.1 Name TWO possible causes. (2)

Networks & WAN

Q56

A family is choosing an internet package.

5.1 Distinguish between a capped and an uncapped account. (2)
5.2 What is the difference between shaped and unshaped bandwidth? (2)
5.3 Distinguish between broadband (fibre) and a mobile (LTE/5G) connection — give ONE advantage of each. (2)

Show memo
  • 5.1 Capped = a limited amount of data per month ✓; uncapped = no data limit (may still be shaped/managed) ✓
  • 5.2 Shaped = the ISP prioritises some traffic and slows others ✓; unshaped = all traffic is treated equally ✓
  • 5.3 Fibre: faster & more stable / lower latency ✓; Mobile: portable / works without a fixed line / quick to set up ✓
Q63

6.2 What is meant by bandwidth throttling? (1)

  • 6.2 The ISP deliberately slowing down a user's connection (e.g. after a usage limit or for certain services) ✓

Practical — Excel, Access, Word & HTML

P1 · Excel — nested IF4

A mark is in B2. In C2 show: "Distinction" for 80+, "Pass" for 50–79, otherwise "Fail". Write the formula.

Show memo 
=IF(B2>=80,"Distinction",IF(B2>=50,"Pass","Fail"))
  • Outer IF for 80+ → "Distinction" ✓
  • Nested IF for 50+ → "Pass" ✓
  • Else "Fail" + brackets close correctly ✓
  • Quotes around all text + correct order (highest first) ✓
P2 · Excel — VLOOKUP & text5

A lookup table of product codes & prices is in F2:G20. A code is typed in A2.

P2.1 Return the matching price for the code in A2. (2)
P2.2 Show the first 3 characters of the code in A2. (1)
P2.3 Count how many cells in A2:A50 are NOT empty. (1)
P2.4 Join the surname in B2 and initial in C2 as "Surname, I". (1)

Show memo
  • P2.1 =VLOOKUP(A2,$F$2:$G$20,2,FALSE) — exact match + absolute table range ✓✓
  • P2.2 =LEFT(A2,3) ✓
  • P2.3 =COUNTA(A2:A50) ✓
  • P2.4 =B2&", "&C2 (or CONCAT) ✓
P3 · Access — advanced query5

Table tblOrders: OrderID, Customer, Amount, OrderDate, Paid (Yes/No).

P3.1 Criterion to show orders with no value captured in Amount. (1)
P3.2 Criterion to show customers whose name starts with "M". (1)
P3.3 In a query, write a calculated field for VAT (15%) on Amount. (1)
P3.4 In a report, which function totals the Amount per group? (1)
P3.5 Criterion to show only unpaid orders. (1)

Show memo
  • P3.1 Is Null ✓
  • P3.2 Like "M*" ✓
  • P3.3 VAT: [Amount]*0.15 ✓
  • P3.4 =Sum([Amount]) (in the group footer) ✓
  • P3.5 No (or False) in the Paid field ✓
P4 · Word — mail merge4

P4.1 What are the TWO documents needed for a mail merge, and what does each contain? (2)
P4.2 Briefly describe how the personalised letters are produced. (2)

Show memo
  • P4.1 The main document — the letter with fixed text + merge fields ✓; the data source — a list of recipients/records (e.g. names & addresses) ✓
  • P4.2 Insert merge fields (e.g. «Name») into the letter ✓; then Finish & Merge → the fields are replaced by each record's data, producing one letter per recipient ✓
P5 · HTML — table formatting4

Adjust an existing table so that: the heading cell spans 2 columns, the border is 2px, and there is space (padding) inside each cell. Name the attributes used.

Show memo
  • Span two columns: colspan="2" on the <td>/<th> ✓
  • Border thickness: border="2" on <table> ✓
  • Space inside cells: cellpadding="..." ✓
  • (Row span equivalent: rowspan; space between cells: cellspacing) ✓

Section C — Integrated Scenario (Q9/Q10 style — apply many topics to one situation)

Scenario: GreenLeaf Nursery goes online

GreenLeaf Nursery is a small family plant business. The owner, Thabo, wants to computerise the shop, take card payments, sell online, and keep records of stock and customers. Answer the sub-questions using everything you know — this is how Section C works.

SC1 · Hardware & setup6

SC1.1 Thabo will photograph every plant for the website. Besides resolution, name TWO camera specifications that affect photo quality. (2)
SC1.2 He wants a device at the counter that reads the price of a plant when its tag is scanned. Name it. (1)
SC1.3 The counter PC must not lose the day's sales if the power trips. Name the device that prevents this and say what it does. (2)
SC1.4 Recommend ONE way to connect the counter PC to the printer without cables. (1)

Show memo
  • SC1.1 Any two: optical zoom, sensor size, aperture/lens quality, ISO range, image stabilisation ✓✓
  • SC1.2 A barcode scanner / barcode reader ✓
  • SC1.3 A UPS (uninterruptible power supply) ✓ — a battery backup that keeps the PC running briefly so work can be saved and the PC shut down safely ✓
  • SC1.4 Bluetooth (or Wi-Fi) ✓
SC2 · Network & internet5

SC2.1 Thabo is choosing between a capped and an uncapped internet account for the shop. Give ONE reason uncapped may suit an online store. (1)
SC2.2 Explain why fibre would be a better choice than mobile data for the shop's fixed connection. (2)
SC2.3 He lets customers use the shop's Wi-Fi. State TWO risks of an open public hotspot. (2)

Show memo
  • SC2.1 An online store uploads images and handles orders all day; uncapped means it won't run out of data / be cut off mid-month ✓
  • SC2.2 Fibre is faster and more stable / has lower latency ✓, and isn't affected by signal strength like mobile data ✓
  • SC2.3 Any two: others can intercept data on the open network; criminals can set up fake hotspots; bandwidth is shared/slowed; malware could spread ✓✓
SC3 · Social, security & data6

SC3.1 GreenLeaf stores customer names, phone numbers and addresses. Name the SA law that requires this data to be protected. (1)
SC3.2 Thabo receives an email that looks like it's from his bank asking him to confirm his password. Name this type of attack and give ONE clue that it is fake. (2)
SC3.3 Give TWO ways Thabo can protect the shop's data against ransomware. (2)
SC3.4 Give ONE green-computing action the nursery can take with its old, broken computers. (1)

Show memo
  • SC3.1 POPIA (Protection of Personal Information Act) ✓
  • SC3.2 Phishing ✓; clue: suspicious sender address, urgent threat, spelling errors, asks for a password/banking details, a link to a fake site ✓
  • SC3.3 Any two: keep regular backups (3-2-1), run updated antivirus, keep software patched, don't open unknown attachments/links ✓✓
  • SC3.4 Take them to a registered e-waste recycler / donate working parts — don't dump them (they contain toxic materials) ✓
SC4 · Info management & practical7

SC4.1 Thabo keeps stock in a database table. Which field would be the best primary key: PlantName or PlantCode? Give a reason. (2)
SC4.2 In a query, what criterion shows only plants with fewer than 5 in stock? (1)
SC4.3 In a spreadsheet, prices are in C2:C40 and a 15% discount is in cell F1. Write a formula for the discounted price of the plant in C2 that can be copied down. (2)
SC4.4 The website links to an online order form. Name the HTML tag AND attribute used to create that link. (2)

Show memo
  • SC4.1 PlantCode ✓ — it is unique for every plant, while two plants could share a name ✓
  • SC4.2 <5 (in the stock/quantity field) ✓
  • SC4.3 =C2*(1-$F$1) — note the absolute reference $F$1 ✓✓
  • SC4.4 The <a> tag with the href attribute ✓✓
Scenario 2: "Bunny Express" food truck

Lerato runs a food truck that moves between office parks. She takes card and phone payments, posts the day's location on social media, and keeps stock in a spreadsheet. This is a second Q10-style scenario — a fresh context, the same skills.

SC5 · Mobile & connectivity5

SC5.1 The truck has no fixed line. Name the type of internet connection Lerato should use and ONE thing that could weaken its signal. (2)
SC5.2 She shares the truck's Wi-Fi with customers. Give ONE reason her mobile data could run out quickly. (1)
SC5.3 Customers tap their bank cards to pay. Name the wireless technology used and explain briefly how the payment works. (2)

Show memo
  • SC5.1 Mobile/cellular data (3G/4G/5G) ✓; signal weakens far from a tower / behind thick walls / in poor-coverage areas ✓
  • SC5.2 Customers streaming/downloading on the shared hotspot use her data allowance ✓
  • SC5.3 NFC ✓; the card and reader exchange data when held very close together, so no PIN/swipe is needed for small amounts ✓
SC6 · Data, social & practical7

SC6.1 Lerato's phone with all her stock records is stolen. Give TWO habits that would have protected her data. (2)
SC6.2 She posts geotagged photos of the truck. Give ONE risk of doing this. (1)
SC6.3 In her spreadsheet, daily sales are in B2:B31. Write a function to show the best single day's sales. (1)
SC6.4 She wants to show only days where sales were below R500 in red, automatically. Name the feature. (1)
SC6.5 She emails a monthly report to 20 suppliers who shouldn't see each other's addresses. Which field should she use? (1)
SC6.6 Give ONE green-computing way she can cut paper when giving receipts. (1)

Show memo
  • SC6.1 Any two: regular backups (cloud/off-device), a screen lock/password, remote-wipe enabled, don't store the only copy on the phone ✓✓
  • SC6.2 Geotags reveal the exact location/routine of the truck, which could aid theft or stalking ✓
  • SC6.3 =MAX(B2:B31) ✓
  • SC6.4 Conditional formatting ✓
  • SC6.5 Bcc (blind carbon copy) ✓
  • SC6.6 Email/SMS digital receipts instead of printing (also: print only on request) ✓

More Exam-Style Questions

More practice across the whole Grade 12 syllabus.

M12

A user only sends email, browses the web and types documents at home. (a) Which user category does she fit? (b) Give ONE reason.

Show memo
  • (a) Home user ✓
  • (b) She only does basic, everyday tasks that don't need high-end hardware ✓
M22

Give TWO reasons an SSD is preferred over an HDD as the main drive in a new laptop.

Show memo
  • Any two: faster data access/boot; no moving parts so more durable (survives bumps); more energy-efficient; runs silently ✓✓
M32

What is metadata? Give an example of metadata for a photo.

Show memo
  • Data about a file (not its content) ✓
  • E.g. date taken, camera model, file size, resolution, GPS location ✓
M42

Distinguish between the read-only and hidden file attributes.

Show memo
  • Read-only = the file can be opened/read but not changed or overwritten ✓
  • Hidden = the file is not shown in File Explorer by default ✓
M52

Distinguish between bandwidth and a data cap.

Show memo
  • Bandwidth = the speed / amount of data that can be transferred per second ✓
  • A data cap = the maximum total amount of data you may use in a month ✓
M62

Distinguish between a browser's cache and cookies.

Show memo
  • Cache stores copies of web page files (e.g. images) so pages load faster on return visits ✓
  • Cookies store small pieces of info about you/your visit (logins, preferences) for a site ✓
M72

What is meant by the 'digital divide', and give ONE way it can be reduced?

Show memo
  • The gap between those who have access to ICT/the internet and those who don't ✓
  • Reduce it by e.g. free public Wi-Fi, cheaper devices/data, community computer centres, ICT in schools ✓
M8 · Excel2

A full name is in cell A2 (e.g. 'Thabo Nkosi'). Write a formula that extracts just the first name.

Show memo
  • =LEFT(A2,FIND(" ",A2)-1) (LEFT + FIND) ✓✓
M9 · Excel2

Distinguish between the COUNT and COUNTA functions in Excel.

Show memo
  • COUNT counts only cells that contain numbers ✓
  • COUNTA counts all non-empty cells (numbers AND text) ✓
M10 · Word2

In Word, distinguish between linking and embedding an object (e.g. an Excel chart).

Show memo
  • Linking keeps a connection to the source file, so the object updates when the source changes ✓
  • Embedding inserts a copy that stays fixed and does not update ✓
M11 · HTML2

In an HTML table, distinguish between colspan and rowspan.

Show memo
  • colspan makes a cell span across two or more columns ✓
  • rowspan makes a cell span down two or more rows ✓
Why Section C feels hard

Each sub-question jumps to a different topic, so you must switch gears fast. The facts are the same ones you already know — the trick is to read what the scenario needs and apply the fact to Thabo's nursery, not answer in general.

How Paper 2 is actually structured

Section A (25): Q1 multiple-choice, Q2 matching, Q3 true/false. Section B (75): Q4 Systems Technologies, Q5 Internet & Network, Q6 Information Management, Q7 Social Implications, Q8 Solution Development. Section C (50): Q9 & Q10 integrated scenarios. Use the topic sets above to drill each part.
