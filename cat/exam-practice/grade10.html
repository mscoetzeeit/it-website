Section A — Objective questions

Paper 2 always opens with multiple choice, matching and true/false. They are quick marks — don't lose them.

A1 · Multiple choice5

1.1 Which device is an input device?  A) Monitor  B) Speaker  C) Scanner  D) Printer

1.2 1 GB is approximately equal to …  A) 1 000 KB  B) 1 000 MB  C) 1 000 TB  D) 100 MB

1.3 Which is an example of system software?  A) Excel  B) Chrome  C) Windows  D) WhatsApp

1.4 The "brain" of the computer that processes instructions is the …  A) RAM  B) CPU  C) HDD  D) GPU

1.5 Which memory is volatile (loses data when power is off)?  A) ROM  B) SSD  C) RAM  D) Flash drive

Show memo
  • 1.1 C) Scanner ✓
  • 1.2 B) 1 000 MB ✓
  • 1.3 C) Windows ✓
  • 1.4 B) CPU ✓
  • 1.5 C) RAM ✓
A2 · Matching4

Match the term in Column A with its description in Column B.

Column A: 1) RAM   2) GUI   3) Driver   4) Peripheral
Column B: a) Software that lets the OS control a device   b) Temporary working memory   c) A device connected to but outside the system unit   d) An interface using windows, icons and a mouse

Show memo
  • 1 → b ✓
  • 2 → d ✓
  • 3 → a ✓
  • 4 → c ✓
A3 · True / False3

State whether each is TRUE or FALSE. If false, give the correct term.

3.1 An SSD has spinning magnetic disks.
3.2 Soft copy is output displayed on a screen.
3.3 ROM is used to store the files you are currently working on.

Show memo
  • 3.1 FALSE — an SSD has no moving parts / uses flash memory (an HDD has spinning disks) ✓
  • 3.2 TRUE ✓
  • 3.3 FALSE — RAM holds the files/programs currently in use ✓

Hardware

Q14

Sipho is buying a new laptop. Distinguish between RAM and ROM by giving TWO differences. (2 × 2)

Show memo

Any two clear differences (1 mark each side):

  • RAM is volatile (loses data when off) ✓ whereas ROM is non-volatile (keeps data) ✓
  • RAM can be read from and written to ✓ whereas ROM is normally read-only ✓
  • (Also accepted) RAM holds data/programs currently in use; ROM holds start-up/BIOS instructions.
Q24

A photographer needs portable storage to back up large video files on location.

2.1 Name a suitable storage device. (1)
2.2 Give ONE reason an SSD is more suitable than an HDD for someone who travels. (1)
2.3 Explain what is meant by storage capacity and name the unit it is measured in. (2)

Show memo
  • 2.1 External SSD / portable SSD / large flash drive / external hard drive ✓
  • 2.2 SSD has no moving parts, so it survives bumps/knocks while travelling (also: faster, lighter) ✓
  • 2.3 Capacity = the amount of data a device can hold ✓; measured in GB / TB (bytes) ✓
Q33

3.1 What is a peripheral device? (1)
3.2 Give ONE advantage and ONE disadvantage of a touchscreen compared to a keyboard and mouse. (2)

Show memo
  • 3.1 A hardware device connected to / controlled by the computer but not part of the core system unit (e.g. printer, mouse) ✓
  • 3.2 Advantage: intuitive / no extra device needed / saves space ✓   Disadvantage: tiring for long use / fingerprints & scratches / less accurate for typing ✓

Software & Licensing

Q44

4.1 Distinguish between system software and application software, giving an example of each. (4)

Show memo
  • System software runs/manages the computer and its hardware ✓, e.g. an operating system like Windows/Android ✓
  • Application software lets the user perform a specific task ✓, e.g. a word processor / Excel / a browser ✓
Q54

A school wants free software for its computers and is comparing licence types.

5.1 Distinguish between freeware and open-source software. (2)
5.2 What is software piracy and why is it illegal? (2)

Show memo
  • 5.1 Freeware = free to use but the source code is not available/cannot be changed ✓; open-source = the source code is available and may be viewed/changed/shared ✓
  • 5.2 Piracy = the illegal copying / distribution / use of software ✓; illegal because it breaks the licence agreement / copyright of the owner ✓
Q63

6.1 State TWO functions of an operating system. (2)
6.2 What is a device driver? (1)

Show memo
  • 6.1 Any two: manages hardware / manages memory / manages files & storage / provides the user interface / manages processes & programs / manages input & output ✓✓
  • 6.2 Software that allows the operating system to communicate with / control a specific hardware device ✓

Networks & Internet

Q74

7.1 What is a computer network? (1)
7.2 Give TWO advantages of connecting the computers in a computer lab to a network. (2)
7.3 Distinguish between a LAN and a WAN. (1)

Show memo
  • 7.1 Two or more computers/devices connected together to share data and resources ✓
  • 7.2 Any two: share files/data, share hardware (e.g. one printer), share an internet connection, communicate easily, central backups ✓✓
  • 7.3 A LAN covers a small area (one building/site) while a WAN covers a large geographical area (e.g. the internet) ✓
Q84

8.1 What does ISP stand for and what does an ISP do? (2)
8.2 Give ONE advantage and ONE disadvantage of a wireless (Wi-Fi) connection compared to a cabled (UTP) connection. (2)

Show memo
  • 8.1 Internet Service Provider ✓ — a company that provides access to the internet (for a fee) ✓
  • 8.2 Advantage of Wi-Fi: no cables / mobile / convenient ✓; Disadvantage: slower / less stable / signal weakens with distance & walls / less secure ✓
Q94

9.1 Give TWO rules of good netiquette. (2)
9.2 What is a search engine? Give an example. (2)

Show memo
  • 9.1 Any two: don't type in CAPITALS (shouting) / be polite & respectful / don't spam / use a clear subject line / don't forward chain mail / think before posting ✓✓
  • 9.2 A website/tool that finds information on the internet using keywords ✓; e.g. Google / Bing / DuckDuckGo ✓

Social Implications

Q104

A learner spends many hours a day at a computer.

10.1 Name TWO health problems that can be caused by poor computer use. (2)
10.2 Give TWO ergonomic ways to prevent these problems. (2)

Show memo
  • 10.1 Any two: eye strain, back/neck pain, repetitive strain injury (RSI)/carpal tunnel, headaches ✓✓
  • 10.2 Any two: use an adjustable/ergonomic chair with good back support, position the screen at eye level & arm's length, take regular breaks & look away, wrist rest / correct keyboard position, good lighting to reduce glare ✓✓
Q114

11.1 What is meant by green computing? (1)
11.2 Give THREE things a computer user can do to be more environmentally friendly. (3)

Show memo
  • 11.1 Using computers/technology in an environmentally responsible way (saving energy & reducing waste) ✓
  • 11.2 Any three: switch off / use sleep mode, print less / print double-sided, recycle e-waste & cartridges, buy energy-efficient (Energy Star) devices, use rechargeable batteries, read on screen instead of printing ✓✓✓
Q123

12.1 What is a digital footprint? (1)
12.2 Give TWO reasons learners should be careful about what they post online. (2)

Show memo
  • 12.1 The trail / record of data you leave behind through your online activity (posts, searches, sites visited) ✓
  • 12.2 Any two: posts are permanent / hard to delete; future employers or schools may see them; could damage your reputation; could expose private information to criminals ✓✓

Practical — Word, Excel & HTML

Paper 1 instructions earn ticks for the correct result. The memos below show the method and the answer.

P1 · Word4

Describe the steps to insert page numbers at the bottom of every page and to add a header with the title "CAT Project".

Show memo
  • Insert tab → Page Number → Bottom of Page → choose a style ✓ (page numbers appear on all pages) ✓
  • Insert tab → Header → choose a style (or double-click top margin) ✓ → type CAT Project ✓
P2 · Excel6

A teacher has marks in cells B2:B21. Write the formula/function for each:

P2.1 The total of all the marks.
P2.2 The average mark.
P2.3 The highest mark.
P2.4 How many learners scored marks (count the numbers).
P2.5 In C2, show "PASS" if B2 is 50 or more, otherwise "FAIL".
P2.6 The lowest mark.

Show memo
  • P2.1 =SUM(B2:B21) ✓
  • P2.2 =AVERAGE(B2:B21) ✓
  • P2.3 =MAX(B2:B21) ✓
  • P2.4 =COUNT(B2:B21) ✓
  • P2.5 =IF(B2>=50,"PASS","FAIL") ✓
  • P2.6 =MIN(B2:B21) ✓

Marks lost for: missing =, wrong range, or missing quotes around text in IF.

P3 · HTML5

Write the HTML to: make a level-1 heading "My Hobbies", a paragraph of text, the word cycling in bold, and a horizontal line under the paragraph. Name the tag used to start a new line.

Show memo 
<h1>My Hobbies</h1>          <-- heading ✓
<p>I enjoy <b>cycling</b> on weekends.</p>  <-- paragraph ✓ + bold ✓
<hr>                              <-- horizontal line ✓

New line tag: <br> ✓

Scenario & Interpretation Practice

Exams love questions where you interpret an advert, a file path or a real situation. Try these.

S1 · Spot the error5

A shop advert reads: Intel Core i5 · 2.4 GHz · 8 TB RAM · 512 GB SSD · 15.6″ screen.

S1.1 One specification has the wrong unit. Which component is it, and what should the unit be? (2)
S1.2 Which item is the CPU, and what does GHz measure? (2)
S1.3 Which component keeps your files when the laptop is switched off? (1)

Show memo
  • S1.1 The RAM — it should be in GB (e.g. 8 GB), not TB ✓✓
  • S1.2 Intel Core i5 ✓; GHz measures the processing speed (clock rate) ✓
  • S1.3 The SSD (storage) ✓
S2 · Read the file path3

A file is stored at: D:\Grade10\Projects\poster.docx

S2.1 What does D: represent? (1)
S2.2 What do Grade10 and Projects represent? (1)
S2.3 Which type of program opens when you double-click poster.docx? (1)

Show memo
  • S2.1 The drive (a storage device/partition) ✓
  • S2.2 Folders — Projects is a subfolder inside Grade10 ✓
  • S2.3 A word processor (e.g. Microsoft Word) ✓
S3 · Mini scenario — the tuck shop5

The school tuck shop uses one computer to record sales and print price lists.

S3.1 Give ONE input and ONE output device the tuck shop would use. (2)
S3.2 The owner wants a backup of the sales file. Give ONE suitable backup medium and ONE reason it is a good choice. (2)
S3.3 Give ONE health tip for the learner who sits at the computer for a long shift. (1)

Show memo
  • S3.1 Input: keyboard / mouse / barcode scanner ✓; Output: monitor / printer ✓
  • S3.2 An external drive / flash drive / the cloud ✓ — because it is portable / has enough space / is stored off-site (safe if the PC is damaged) ✓
  • S3.3 Take regular breaks & stretch / sit up straight / screen at eye level / look away often ✓

Your Grade 10 class is hosting a market day to raise funds. You help with the computer side.

S4.1 The class uses one laptop and a printer to make posters. Give ONE input and ONE output device used. (2)

S4.5 A learner posts the event on social media. Give ONE netiquette rule she should follow. (1)

Show memo
  • S4.1 Input: keyboard / mouse / touchpad ✓; Output: printer / monitor ✓

  • S4.3 A watermark ✓
  • S4.4 =SUM(B2:B10) ✓
