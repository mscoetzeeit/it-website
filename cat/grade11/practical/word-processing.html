Word Processing Grade 11

Grade 11 builds on Grade 10 Word skills. New topics include templates, sections, advanced styles, mail merge, references (TOC, footnotes, captions), electronic forms and document themes.

File Management — Extended

Templates

A template is a pre-designed document with built-in formatting and layout. New documents based on the template inherit all its formatting.

  • Use a template: File → New → search for a template or select one from the gallery
  • Create a template: design your document → File → Save As → change file type to Word Template (.dotx)
  • Templates save time for letters, invoices, reports and certificates

Import/Input Data

  • Insert content from other formats: Insert → Object, or copy and paste from Excel/CSV/text files
  • Paste Special (Ctrl+Alt+V): choose HOW to paste — Unformatted Text, Keep Source Formatting, Link, Paste as Picture

Page Layout and Design

Themes

A theme applies a coordinated set of colours, fonts and effects to the entire document: Design tab → Themes.

Cover Page

Insert → Cover Page → choose from built-in designs. The cover page uses content controls for title, subtitle, author, etc.

Advanced Document Layout

Sections and Section Breaks

Sections allow different formatting in different parts of the same document (e.g. different headers, orientation or column layout).

  • Insert a section break: Layout → Breaks → Section Breaks
  • Next Page: new section starts on next page
  • Continuous: new section continues on same page
  • Odd/Even Page: new section on next odd or even page

Headers and Footers — Advanced

In different sections you can have different headers/footers. Fields you can insert:

  • Date: current date (auto-updates)
  • Author: the document creator's name
  • File path and name: shows where the file is saved
  • Document title: the title set in document properties

Advanced Page Numbering

  • Different first page numbering (cover page has no number)
  • Odd/even different page numbers (like a book)
  • Start numbering from a specific number (e.g. start from page 3)
  • Different numbering formats: 1 2 3 / i ii iii / A B C

Columns

  • Layout → Columns → choose number of columns
  • Options: number of columns, spacing between columns, line between columns
  • Column break: forces text to start in the next column (Layout → Breaks → Column)

Paragraph Formatting — Advanced

Customise Bullets and Numbering

  • Click dropdown on Bullets/Numbering button → Define New Bullet/Number Format
  • Change bullet symbol (any character from any font), colour and size
  • Outline Numbering / Multi-level Lists: creates numbered lists with sub-levels (1, 1.1, 1.1.1)

Drop Cap

A drop cap is a large capital letter at the start of a paragraph that drops down across several lines — the decorative big letter you see at the start of chapters in magazines and storybooks.

Step-by-step: Add a dropped cap, 3 lines high, 0.5 cm from the text
  1. Click anywhere inside the paragraph you want to start with a drop cap. (You do not need to highlight anything — just click in the paragraph so the cursor is blinking in it.)
  2. At the top of the screen, click the Insert tab.
  3. Look for the section called Text group (towards the right-hand side of the ribbon).
  4. Click the Drop Cap button (its icon is a large letter "A").
  5. A small menu drops down. Click Drop Cap Options… at the bottom (do not just click "Dropped" yet — we want to set the exact size).
  6. A dialog box opens. Under Position, click the picture labelled Dropped.
  7. In the Lines to drop box, delete the number and type 3.
  8. In the Distance from text box, type 0.5 cm.
  9. Click OK. The first letter of the paragraph is now large and drops down 3 lines.

Tables — Advanced Formulas

In Word tables you can use basic formulas (Table Layout → Formula):

FormulaEffect
=SUM(ABOVE)Adds all values in cells above the current cell
=SUM(LEFT)Adds all values in cells to the left
=AVERAGE(ABOVE)Average of cells above
=COUNT(LEFT)Count of cells to the left
=MAX(ABOVE)Largest value above
=MIN(ABOVE)Smallest value above

Styles — Advanced

Styles apply consistent formatting and are essential for automatic Table of Contents generation.

  • Apply a style: select text → click style in the Styles gallery (Home tab)
  • Modify a style: right-click style in gallery → Modify → change font, size, colour, etc. All text using that style updates automatically
  • Create a new style: Home → Styles dialog → New Style button → define name, formatting and base

References

Table of Contents (TOC)

A TOC is generated automatically from Heading styles: References → Table of Contents → choose a style. Update it with: References → Update Table.

Footnotes and Endnotes

  • Footnote: reference note at the bottom of the current page
  • Endnote: reference note at the end of the document
  • Insert: References → Insert Footnote / Insert Endnote

Captions

Labels for figures, tables and images: References → Insert Caption. Captions auto-number (Figure 1, Figure 2, etc.) and can generate a List of Figures.

Citations and Bibliography

References → Manage Sources → add your sources → References → Bibliography → generate automatically formatted bibliography (APA, MLA, etc.)

Index

Mark entries (References → Mark Entry) then insert an index (References → Insert Index) at the end of the document.

Electronic Forms (Legacy Tools)

Forms allow users to fill in specific fields without editing the rest of the document.

  • Developer tab (enable in File → Options → Customize Ribbon)
  • Legacy Tools: Text Form Field, Check Box, Drop-Down Form Field
  • Protect the document: Developer → Restrict Editing → allow only form filling

Mail Merge

Mail merge creates personalised copies of a document for multiple recipients (e.g. 30 personalised letters from one template).

Steps (Mailings tab):

  1. Start Mail Merge → Letters (or Labels, Emails, etc.)
  2. Select Recipients → choose data source (Word table, Excel spreadsheet, CSV file, Access database)
  3. Write your document → insert Merge Fields where the personalised data should go (e.g. <<FirstName>>, <<Address>>)
  4. Preview Results → check how each letter looks
  5. Finish & Merge → Edit Individual Documents (creates a new document with all merged letters) OR Print Documents
EXAM TIP — Mail Merge Data Sources

The data source can be: a Word table, an Excel spreadsheet, a CSV file (comma-separated), or an Access database table. In Grade 11, you use Word tables or spreadsheets. In Grade 12, you extend to all sources including email lists.
