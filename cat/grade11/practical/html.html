HTML & Web Design Grade 11

Grade 11 extends on Grade 10 HTML basics. New topics: HTML tables, lists (ordered and unordered), images, hyperlinks, comments, and basic colour use in web design.

Revision — HTML Structure

All Grade 10 tags still apply. Grade 11 builds on top of them.

HTML Comments

Comments are notes for the developer — they are NOT displayed in the browser.

Comment syntax
<!-- This is a comment. The browser ignores this. -->
<p>This text IS displayed.</p>

Attributes — Extended

Attributes control the appearance and behaviour of elements. Multiple attributes can be used in one tag:

Multiple attributes
<p align="center">Centred paragraph</p>
<h2 align="right">Right-aligned heading</h2>

HTML Lists

Unordered (Bulleted) List

Unordered list
<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

<!-- type attribute changes bullet symbol -->
<ul type="disc">    <-- ● (default)
<ul type="circle">  <-- ○
<ul type="square">  <-- ■

Ordered (Numbered) List

Ordered list
<ol>
  <li>First step</li>
  <li>Second step</li>
  <li>Third step</li>
</ol>

<!-- type attribute changes numbering style -->
<ol type="1">   <-- 1, 2, 3 (default)
<ol type="A">   <-- A, B, C
<ol type="a">   <-- a, b, c
<ol type="I">   <-- I, II, III (Roman numerals)
<ol type="i">   <-- i, ii, iii

HTML Images

Image tag
<img src="logo.png" alt="Company Logo">

<!-- With width and height -->
<img src="photo.jpg" alt="School photo" width="300" height="200">

<!-- With border -->
<img src="icon.gif" alt="Icon" border="2">

HTML Tables

Table structure
<table border="1">
  <tr>                        <!-- Table Row -->
    <th>Name</th>              <!-- Table Header (bold, centred) -->
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Amahle</td>            <!-- Table Data (regular cell) -->
    <td>17</td>
    <td>Cape Town</td>
  </tr>
  <tr>
    <td>Johan</td>
    <td>18</td>
    <td>Pretoria</td>
  </tr>
</table>
How to think about building a table (type it in this order)
  1. Start with the table "fence": type the opening <table border="1"> and, a few lines below, the closing </table>. Everything else goes between them.
  2. Each ROW is a <tr> … </tr> pair. Add one <tr></tr> for every row you need.
  3. Inside each row, add the CELLS. Use <th> for heading cells (bold, centred) in the first row, and <td> for normal data cells in the other rows.
  4. Type your text between each cell's tags, e.g. <td>Amahle</td>
  5. Save (Ctrl + S) and refresh the browser (F5) to check it. Count your <td> tags — every row should have the same number of cells.
<table>
Creates the entire table
<tr>
Table Row — one horizontal row
<th>
Table Header cell — bold and centred by default
<td>
Table Data cell — regular content cell

Table Attributes

AttributeApplied toEffect
border<table>Thickness of the table border (e.g. border="1")
cellpadding<table>Space INSIDE each cell between content and border
cellspacing<table>Space BETWEEN cells
width<table> or <td>Width of the whole table or a specific cell
bgcolor<table>, <tr>, or <td>Background colour
align<td> or <th>Horizontal alignment: left, center, right
valign<td> or <th>Vertical alignment: top, middle, bottom
colspan<td> or <th>Merges cells horizontally across columns

Merging cells — colspan and rowspan
<!-- colspan: merge across 2 columns -->
<td colspan="2">This cell spans two columns</td>

<!-- rowspan: merge across 3 rows -->
<td rowspan="3">This cell spans three rows</td>

Hyperlinks

Anchor tag — hyperlinks
<!-- Link to a website -->
<a href="https://www.google.com">Go to Google</a>

<!-- Link to another page in same folder -->
<a href="page2.html">Go to Page 2</a>

<!-- Open link in new tab -->
<a href="https://dbe.gov.za" target="_blank">DBE Website</a>

<!-- Link to a bookmark (id) on the same page -->
<a href="#section2">Jump to Section 2</a>
<!-- The target bookmark: -->
<h2 id="section2">Section 2</h2>

<!-- Link to an email address -->
<a href="mailto:teacher@school.co.za">Email us</a>

<!-- Image as a link -->
<a href="home.html"><img src="logo.png" alt="Home"></a>
target ATTRIBUTE VALUES

target="_blank" — opens in a new browser tab.
target="_self" — opens in the same tab (default).

Good Website Design Principles

  • Readability: clear font, appropriate size, sufficient contrast between text and background
  • Navigation: easy to find links; consistent menu position; clear labels
  • Consistency: same colours, fonts and layout on all pages
  • Layout: logical structure; visual hierarchy (headings, sub-headings, body text)
  • Typography: limit to 2 font families; body text 12–14pt; headings larger
  • Colour use: limit to 2–3 main colours; use complementary colours; avoid clashing colours
EXAM TIP — Cite Tag (PAT Only)

The <cite> tag is used in CAT only for the PAT (not in exam P1). It marks a citation or reference to another work. It typically renders in italic.
Example: <cite>DBE CAPS Document, 2023</cite>
