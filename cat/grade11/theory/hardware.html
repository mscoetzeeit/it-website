Hardware & Processing Grade 11

Grade 11 extends hardware knowledge — covering processing components in detail, monitor/printer quality factors, input/output devices for special contexts (POS, ATM, disabilities) and cloud storage.

Wearables

Wearable technology includes smartwatches, fitness trackers, smart glasses and medical devices:

  • Collect health/activity data (heart rate, steps, sleep)
  • Display notifications from phone
  • GPS location tracking
  • Etiquette: put smart devices on silent in meetings; avoid filming people without permission

Output Devices — Quality Factors

Monitors

Resolution
Number of pixels (dots) — e.g. 1920×1080 (Full HD). Higher resolution = sharper, clearer image
Aspect ratio
Width:Height ratio — e.g. 16:9 (widescreen) or 4:3 (older screens)
Colour depth
Number of colours the monitor can display — more bits = more colours (32-bit = over 4 billion colours)
Size
Measured diagonally in inches; larger = more workspace

Printers

Resolution (dpi)
Dots per inch — more dpi = sharper print quality
ppm
Pages per minute — how fast the printer prints
Duty cycle
Maximum number of pages the printer is designed to print per month

Processing — The System Unit in Detail

Motherboard
Main circuit board that houses and connects all internal components (CPU, RAM, expansion slots)
CPU (Central Processing Unit)
Carries out all processing (calculations, running programs). Speed in GHz. Multiple cores allow parallel processing.
GPU (Graphics Processing Unit)
Dedicated processor for graphics and video; either integrated (on motherboard) or dedicated (separate card)
RAM (Random Access Memory)
Holds data and instructions during processing; volatile; the more RAM, the more programs can run simultaneously
ROM (Read Only Memory)
Stores start-up instructions (BIOS/UEFI); non-volatile; cannot normally be changed
NIC (Network Interface Card)
Connects the computer to a network (wired Ethernet or wireless Wi-Fi)

Storage — Primary vs Secondary

Primary Storage (Memory)Secondary Storage
ExamplesRAM, ROM, CacheHDD, SSD, Flash drive, DVD
Volatile?RAM: Yes (lost on power off)No — data persists

Cloud Storage

Cloud storage saves files on remote servers accessed via the Internet, rather than on local storage devices.

CLOUD STORAGE = SOFTWARE AS A SERVICE (SaaS)

Cloud storage is a software service (SaaS) even though it involves hardware and networks. Examples: Google Drive, OneDrive, Dropbox, iCloud.

AdvantagesDisadvantages
Access files from any device/locationRequires Internet connection
Automatic backup of filesMonthly subscription costs (for large storage)
Easily share files/collaboratePrivacy and security concerns (data stored by 3rd party)
No physical storage device neededSpeed depends on Internet bandwidth
Scales up easily (buy more space)Data can be lost if provider closes/is hacked
EXAM TIP — Interpreting Adverts

In Grade 11, you may be given a hardware advert and asked to interpret it. Know what these specifications mean: CPU (e.g. Intel Core i7 at 3.6 GHz), RAM (e.g. 16 GB DDR4), Storage (e.g. 512 GB SSD + 1 TB HDD), GPU (e.g. 4 GB dedicated NVIDIA), monitor resolution (e.g. 1920×1080). Be able to say which user type this is suitable for.

Common Exam Mistakes

RAM stores files permanently.
RAM is temporary (volatile) — its contents are lost when the power is off. Storage (HDD/SSD) keeps files.
A scanner is an output device.
A scanner is an input device — it captures an image or text into the computer.
A bigger hard drive makes the computer faster.
More storage = more space, not more speed. More RAM or an SSD improves speed.
