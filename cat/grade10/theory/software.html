Software & Licensing Grade 10

Software is the set of programs and instructions that tell hardware what to do. This topic covers the types of software, licensing agreements, and how operating systems work.

What is Software?

Software = programs and instructions that tell the hardware how to operate. Without software, hardware is just an expensive collection of metal and plastic. Software is non-physical — you cannot touch it.

System Software vs Application Software

System Software in Detail

Operating System (OS)

The operating system manages all hardware and software on the computer. It provides the user interface (GUI or command line) and coordinates all tasks.

Device TypeCommon Operating Systems
Desktop / LaptopWindows 10/11, macOS, Linux (Ubuntu)
Smartphone / TabletAndroid (Google), iOS (Apple)

Drivers

A driver is a small program that allows the operating system to communicate with a hardware device. When you plug in a new printer or mouse, the OS installs a driver for it. Without the correct driver, the device will not work.

Auto Configuration (Plug-and-Play)

Plug-and-Play (PnP) means the computer automatically detects and configures a new device when you connect it — without needing to manually install drivers or restart.

Utility Programs

Utility programs help maintain and optimise the computer:

  • Antivirus — detects and removes malware
  • Disk Cleanup — removes temporary files to free up space
  • Defragmenter — reorganises HDD files for faster access
  • File compression — reduces file sizes (e.g. WinZip, 7-Zip)
  • Backup software — creates copies of important data
  • Firewall — monitors network traffic to block threats

Software Licensing

KEY DEFINITION

A software licence is a legal agreement between the developer and the user that states how the software may be used. Buying software does not mean you own it — you only buy the right to use it.

Types of Licence Agreements

EULA
End User Licence Agreement — standard single-user licence; you agree to terms when you install the software
Site Licence
Allows installation on multiple computers in one location (e.g. a school may buy a site licence for Office)
Creative Commons
Used for creative works (images, music, documents); creator chooses what permissions to grant others

Software Piracy and Copyright

ILLEGAL

Software piracy = copying, distributing or using software without a valid licence. This is illegal and unethical. It violates copyright law. Consequences include fines and criminal prosecution.

  • Copyright = legal protection given to creators over their original work. Software is automatically copyrighted.
  • Intellectual property = creations of the mind (music, art, software, inventions) protected by law.
  • Buying a software licence does NOT give you ownership — it gives you a right to use it under the specified conditions.

EXAM TIP — Open-source vs Proprietary

Know the advantages of open-source: free, community-supported, customisable, transparent code. And the disadvantages: less professional support, can be more difficult to use, security depends on community vigilance. Proprietary software advantage: professional support and regular updates; disadvantage: expensive.
