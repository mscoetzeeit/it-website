Internet & E-Communication (T3) Grade 10

The Internet is the world's largest network. This topic covers the Internet, the World Wide Web, browsers, search engines and electronic communication including email and netiquette.

Term 3

The Internet vs the WWW

InternetWorld Wide Web (WWW)
What is it?A global network of interconnected computers and devicesA collection of websites and web pages accessed via the Internet
Older/newer?Older — the infrastructureNewer — a service that runs on the Internet
AnalogyLike the road systemLike the shops on the roads
ProtocolTCP/IPHTTP / HTTPS
KEY POINT

The Internet is the physical infrastructure (cables, servers, routers). The WWW is a service that uses the Internet. Email, online gaming and video streaming also use the Internet but are NOT part of the WWW.

Term 3

Internet Addresses

IP Address
Unique numerical address assigned to each device on a network (e.g. 192.168.1.1). Identifies where data should be sent.
URL
Uniform Resource Locator — the full address of a specific web page (e.g. https://www.google.com/search)
URL Shortener
A service that converts a long URL into a short one (e.g. bit.ly). Useful for sharing links.
Domain Name
Human-friendly name for a website (e.g. google.com). Converted to an IP address by a DNS server.

The Parts of a URL

https://www.capeschools.co.za/grade10/index.html
protocol  www (host)  domain name  path to the page/file
HTTP vs HTTPS

http:// is unencrypted; https:// is secure — data is encrypted (look for the padlock 🔒). Always check for HTTPS before entering passwords or banking details.

Term 3

Web Browsers

A browser is software used to access and display web pages. It requests pages from web servers using HTTP/HTTPS.

Chrome
Google's browser — most popular globally
Edge
Microsoft's browser — built into Windows
Firefox
Mozilla's open-source browser
Safari
Apple's browser — default on Mac and iPhone

Tabbed browsing: allows multiple web pages to be open simultaneously in separate tabs within one browser window — more efficient than opening multiple windows.

Term 3

Search Engines

A search engine is a website/tool that searches the Internet for information based on keywords you type. It maintains an index of billions of web pages.

  • Google — most popular search engine
  • Bing — Microsoft's search engine
  • Yahoo — older search engine/portal
Term 3

Downloading vs Uploading

Downloading
Receiving data FROM the Internet TO your device (e.g. saving a file, streaming a movie to watch)
Uploading
Sending data FROM your device TO the Internet (e.g. posting a photo on Instagram, submitting a form)

Term 3

Synchronous vs Asynchronous Communication

E-communication can be grouped by when the people involved take part:

  • Synchronous communication happens in real time - everyone takes part at the same moment (e.g. a video call or live chat).
  • Asynchronous communication does not happen at the same time - the receiver replies whenever it suits them (e.g. email or SMS).

To, Cc and Bcc — a favourite exam question

FieldWho it's forCan other recipients see this address?
ToThe main recipient(s) of the messageYes
Cc (Carbon copy)People who should get a copy for informationYes — everyone sees Cc names
Bcc (Blind carbon copy)Copy someone privatelyNo — Bcc addresses are hidden from everyone
EXAM TIP — when to use Bcc

Use Bcc when emailing many people who don't know each other (e.g. a newsletter), to protect their privacy and hide their email addresses from one another.

Term 3

Netiquette (Internet Etiquette)

Netiquette = good manners and correct behaviour when communicating online. Rules include:

  • Do NOT type in ALL CAPS — it looks like you are SHOUTING
  • Check your spelling and grammar before sending
  • Be concise and courteous — do not write unnecessary long messages
  • Do not gossip or share private information about others
  • Compress attachments before sending to reduce file size
  • Use a clear and relevant subject line
  • Do not forward chain emails or spam
  • Respect others' privacy — do not share their email addresses without permission
Term 3

Safe Internet Use

  • Multi-level authentication: use more than one method to verify identity (password + OTP)
  • OTP (One-Time Password): a unique code sent to your phone for a single login attempt
  • Never share your passwords with anyone
  • Be suspicious of unexpected emails with links or attachments (phishing)
  • Only visit secure websites (HTTPS, padlock icon in browser)

Common Exam Mistakes

A web browser and a search engine are the same.
A browser (Chrome, Edge) displays web pages; a search engine (Google) finds pages by keyword.
Cc and Bcc do the same thing.
Cc addresses are visible to all; Bcc addresses are hidden from everyone.
HTTP and HTTPS are equally safe.
HTTPS encrypts the data (padlock); HTTP does not.
