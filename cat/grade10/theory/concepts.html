Concepts of Computing Grade 10

This topic covers the fundamentals: what a computer is, types of computing devices, ICTs, and the difference between data and information.

What is a Computer?

A computer is an electronic device that accepts data (input), processes it according to a set of instructions (program), stores the results and produces output. The basic model is the Information Processing Cycle:

INPUT
→
PROCESSING
→
OUTPUT
↕
STORAGE
↔
COMMUNICATION

Types of Computing Devices

Multi-purpose Devices

DeviceDescription / Example

Dedicated Devices (single-purpose)

Dedicated devices have an embedded computer designed for one specific function:

  • ATM — dispenses cash, checks balances
  • Digital camera — captures photos/video
  • Printer — prints documents
  • Fitness tracker / Smartwatch — monitors health data
  • Factory robots — automated manufacturing
  • Smart appliances — fridges, microwaves with embedded chips
CONCEPT OF CONVERGENCE

Convergence is when multiple functions are combined into one device. A smartphone is the best example — it is a phone, camera, GPS navigator, music player, internet browser and more, all in one device.

What is ICT?

ICT (Information and Communication Technology) refers to technologies used to handle, store, retrieve, send and receive information. An ICT system has three core functions:

Convey
Transfer/communicate data between devices or people (e.g. sending an email, making a call)
Manipulate
Process and change data (e.g. calculating totals in a spreadsheet)
Store
Save data for later retrieval (e.g. saving a file to a hard drive or cloud)

ICTs in Everyday Life

  • Point of Sale (POS) system: barcode scanner + computer + card machine at a shop checkout
  • Cell phone system: phone + cell towers + network infrastructure
  • Online banking: computer/phone + internet + bank servers
  • School management system: stores learner records, attendance, marks

Data vs Information

KEY DISTINCTION

Data = raw, unprocessed facts with no context (e.g. the numbers 85, 72, 90).
Information = data that has been processed and given meaning (e.g. "Learner achieved an average of 82% in CAT").
Remember: Data → processed → Information → used to make decisions (Knowledge).

The GUI (Graphical User Interface)

A GUI is a visual interface that allows users to interact with the computer using icons, windows, menus and a mouse — instead of typing text commands. All modern operating systems (Windows, macOS, Android, iOS) use a GUI.

Desktop GUI features (Windows):

  • Desktop — the working area (like a physical desk)
  • Icons — small pictures representing programs or files
  • Taskbar — bar at the bottom showing open programs and the Start button
  • Start button — opens the menu to access all programs
  • Recycle Bin — holds deleted files until permanently removed
  • Shortcuts — quick access icons placed on the desktop
  • File Manager (This PC / Windows Explorer) — browse folders and files
EXAM TIP

Know the difference between data and information — this is a very common exam question. Always remember: data is raw, information is meaningful. Also be able to give examples from a school context (e.g. test marks = data; "learner passed Term 1" = information).
