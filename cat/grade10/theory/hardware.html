Hardware Grade 10

Hardware is every physical component of a computer that you can touch. It includes input devices, output devices, storage devices and the processing components inside the system unit.

What is Hardware?

Hardware refers to the physical parts of a computer system. It works together with software (programs/instructions) to perform tasks. Hardware without software is useless, and software cannot run without hardware.

KEY DEFINITION

Hardware = any physical component of a computer that you can touch and see. Peripheral devices = hardware devices connected to (but not part of) the main system unit, e.g. keyboard, mouse, printer.

EXAM TIP — Advantages vs Limitations

Examiners often ask you to state the advantage OR limitation of a specific input device. Always link your answer to the context given in the question (e.g. a touchscreen is convenient but scratches easily; a barcode reader is fast and accurate but cannot read damaged barcodes).

Hard Copy vs Soft Copy

Hard copy
Printed physical output on paper — permanent and tangible (e.g. printed document, photograph)
Soft copy
Digital output displayed on a screen — temporary and electronic (e.g. document on monitor, PDF on tablet)

Storage

Storage saves data permanently (even when power is off), unlike RAM which is temporary. Storage capacity is measured in KB → MB → GB → TB.

COMPARING STORAGE MEDIA

When comparing storage, consider: Capacity (how much can it hold?), Speed (how fast does it read/write?), Robustness (how well does it withstand damage?), Volatility (does it lose data when power is cut?), Physical size (is it portable?), Cost.

Processing — Inside the System Unit

Motherboard
Main circuit board that connects all components together (CPU, RAM, expansion cards)
CPU (Central Processing Unit)
The "brain" of the computer — carries out program instructions; speed measured in GHz
RAM (Random Access Memory)
Temporary storage — holds data and programs currently in use; volatile (data lost when power off); measured in GB
ROM (Read Only Memory)
Permanent memory with start-up instructions (BIOS); non-volatile ; cannot normally be changed.

EXAM TIP — Peripheral Devices

A peripheral device is any hardware device that connects to and is controlled by the computer but is not part of the core system unit. Examples: keyboard, mouse, printer, scanner, webcam. Remember: monitors ARE considered peripheral devices too.

Common Exam Mistakes

RAM stores files permanently.
RAM is temporary (volatile) — its contents are lost when the power is off. Storage (HDD/SSD) keeps files.
A scanner is an output device.
A scanner is an input device — it captures an image or text into the computer.
A bigger hard drive makes the computer faster.
More storage = more space, not more speed. More RAM or an SSD improves speed.
