Spreadsheets Grade 10

Microsoft Excel is the spreadsheet application used in CAT. Spreadsheets organise data into rows and columns and allow powerful calculations, charts and data analysis.

The Excel Workspace

Workbook
The entire Excel file (e.g. Sales.xlsx); contains one or more worksheets
Worksheet / Sheet
A single tab/page within the workbook; made up of rows and columns
Column
Vertical — labelled A, B, C … Z, AA, AB…
Row
Horizontal — labelled 1, 2, 3…
Cell
The intersection of a column and row (e.g. B3)
Active cell
The currently selected cell — shown in the Name Box (top left)
Name Box
Shows the address of the active cell; click to navigate to a cell
Formula Bar
Shows the content (value or formula) of the active cell; edit here

Cell References

A cell reference identifies a specific cell by its column letter and row number. Using cell references in formulas is critical — if the data changes, the formula automatically recalculates.

WHY USE CELL REFERENCES?

Instead of typing =50+30, use =B2+C2. If the values in B2 or C2 change, the answer updates automatically. Hard-coded numbers never update.

Range Names

A named range gives a meaningful name to a group of cells. Select the cells → Name Box → type name → Enter. Then use the name in formulas:

Example
=SUM(Marks)   ← instead of =SUM(B2:B20)

Basic Calculations

All formulas start with =. Order of operations (BODMAS) applies.

OperatorMeaningExample

-Subtraction=A1-B1
*Multiplication=A1*B1
/Division=A1/B1
^Exponent (power)=A1^2 (A1 squared)
( )Brackets — calculate first=(A1+B1)*C1

Basic Functions

Functions are built-in formulas. Syntax: =FUNCTIONNAME(argument1, argument2, …)

Step-by-step: Add up a column of numbers with SUM
  1. Click the empty cell where you want the answer to appear (for example, the cell just below your list of numbers).
  2. Type an equals sign =. Every formula MUST start with =, otherwise Excel treats it as plain text.
  3. Type the word SUM followed by an opening bracket: =SUM(
  4. Now select the numbers: click the first number cell, hold the mouse button and drag down to the last number, then let go. Excel fills in the range for you, e.g. =SUM(B2:B10
  5. Type a closing bracket ) so it reads =SUM(B2:B10)
  6. Press Enter. The cell now shows the total. (Click the cell again and look at the Formula Bar at the top — it still shows the formula, not the answer.)
Try it yourself

Type the numbers 10, 20 and 30 into cells A1, A2 and A3. In cell A4 type =SUM(A1:A3) and press Enter . You should see 60 to see. Now change A1 to 100 — watch A4 update to 150 automatically. That is the power of using cell references!

COUNTIF — Relational Operators

COUNTIF criteria use relational operators inside quotes: ">50", "<=30", "=Pass", "<>0" (not equal to 0).

Formatting Cells

Number Formats (Home → Number group or Ctrl+1)

  • General — no specific format
  • Number — set decimal places (e.g. 2)
  • Currency — adds R or $ symbol
  • Percentage — multiplies by 100 and adds % sign
  • Date — displays numbers as dates

Cell Formatting

  • Borders, fill colour (background), alignment (left/centre/right/wrap text)
  • Merge & Centre — combines selected cells into one (great for headings)
  • AutoFill — drag the fill handle (bottom-right corner of cell) to fill a pattern (1,2,3… or Jan, Feb, Mar…)

Rows and Columns

  • Resize: drag the column/row border, or right-click → Column Width / Row Height
  • Insert: right-click row/column header → Insert
  • Delete: right-click → Delete
  • Hide/Unhide: right-click → Hide / Unhide

AutoFill

AutoFill saves you from typing repetitive or predictable data by hand — like a list of dates, months, or numbers counting up — by recognising the pattern from one or two cells and continuing it for you.

Step-by-step: Use AutoFill
  1. Type the starting value (e.g. Monday , or 1 and 2 in two cells to show the pattern).
  2. Select the cell(s), then hover over the small green square at the bottom-right corner of the selection — the cursor turns into a thin black +.
  3. Click and drag down or across over the cells you want to fill. Excel continues the pattern (days, months, or counting numbers).
  4. To copy a formula instead of a list, AutoFill the same way — Excel automatically adjusts relative cell references for each new row.
AUTOFILL OPTIONS BUTTON

After dragging, a small button appears near the filled cells — click it to choose "Copy Cells" (repeats the exact same value) instead of "Fill Series" if you didn't want the pattern continued.

Sorting

Sort data alphabetically or numerically: Data → Sort. Choose the column to sort by and Ascending (A→Z, 0→9) or Descending (Z→A, 9→0).

Error Indicators

ErrorCause

#NAME! Function name is misspelled (e.g. =SUMM())
#DIV/0!Formula is trying to divide by zero
#REF!A cell reference is invalid — a referenced cell was deleted
#VALUE!Wrong type of data (e.g. trying to add text to a number)
#NUM!Invalid numeric value (e.g. square root of a negative number)

Charts and Graphs (Term 4)

Select the data → Insert → Charts → choose chart type.

Chart TypeBest used for
Column / BarComparing values across categories
PieShowing parts of a whole (percentages); works with one series only
LineShowing trends over time

After inserting a chart, use Chart Design and Format tabs to: add chart title, axis labels, data labels, change colours, move the chart.

EXAM TIP

Common exam mistakes: forgetting the = sign before a formula; using a comma instead of a colon in ranges (B2,B10 instead of B2:B10 ); selecting the wrong range. Always double-check your formula in the Formula Bar before moving on.
