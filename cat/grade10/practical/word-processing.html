Word Processing Grade 10

Microsoft Word is the word processing application used in CAT. Grade 10 covers all the essentials — from creating documents and formatting text to working with tables, page layout and objects.

HOW TO READ THE STEPS ON THIS PAGE

When you see a path like Home tab→Font group→Bold it means: click the word Home at the top of the screen, then look in the section called Font, then click the Bold button. Keys you press on the keyboard look like this: Ctrl + S (hold the first key down, then tap the second).

Document Structure

A Word document is made up of:

  • Pages / Sections — the physical pages; sections allow different formatting in different parts
  • Paragraphs — blocks of text ending with a paragraph mark (¶)
  • Lines / Text — individual rows of characters
  • Objects — images, shapes, text boxes, charts inserted into the document
FORMATTING MARKS

Turn on formatting marks with Ctrl+Shift+8 or the ¶ button on the Home tab. This shows paragraph marks (¶), spaces (·) and tab characters (→) — very helpful for troubleshooting layout problems.

TYPOGRAPHY TIP

Use only ONE space after punctuation (full stops, commas, etc.) — NOT two spaces. Word's autocorrect will also fix "smart quotes" (curly ") and proper em-dashes (—) automatically.

Step-by-step: Make a heading bold, size 16, dark blue
  1. Move your mouse to the very start of the text you want to change. Press and hold the left mouse button, then drag across the words to highlight (the text turns a shaded colour). Let go of the button.
  2. At the top of the screen, click the Home tab (it is usually already open).
  3. Find the section labelled Font near the top-left. Click the small arrow next to the font size box (it shows a number like 11) and choose 16.
  4. Still in the Font group, click the B button to make the text bold. (Or press Ctrl + B.)
  5. Click the small arrow next to the A with the coloured line under it (the Font Colour button).
  6. A grid of colours appears. Click a dark blue square. If you cannot see the exact colour, click More Colours… and pick one.
  7. Click anywhere else on the page to deselect, and check your heading looks correct.

Paragraph Formatting (Home Tab → Paragraph Group)

Alignment

Align Left
Ctrl+L — text lines up against the left margin (default)
Centre
Ctrl+E — text is centred between the margins
Align Right
Ctrl+R — text lines up against the right margin
Justify
Ctrl+J — text stretches to fill the line between both margins

Spacing

  • Line spacing: controls space between lines within a paragraph (1.0 = single, 1.5, 2.0 = double)
  • Paragraph spacing: adds space Before or After a paragraph (set in Paragraph dialog: Layout → Paragraph)

Indents

  • Increase indent: Tab / Ctrl+M — moves paragraph to the right
  • Decrease indent: Shift+Tab / Ctrl+Shift+M — moves paragraph back left
  • First line indent: only the first line of the paragraph is indented
  • Hanging indent: first line is at the margin; all other lines are indented

Bullets and Numbering

  • Bullets: unordered list with symbols (•, ◆, ✓) — use when order doesn't matter
  • Numbering: ordered list (1, 2, 3 or a, b, c) — use when sequence matters

Tabs

Document Layout

Page Numbers
Insert → Page Number — adds automatic page numbers (top, bottom, margin)
Page Break
Ctrl+Enter — forces the next content to start on a new page
Headers & Footers
Text that appears at the top (header) or bottom (footer) of every page — e.g. title, date, page number
Step-by-step: Put your name in the header and a page number in the footer
  1. Click the Insert→Header button, then click Edit Header (or pick a built-in design). The cursor jumps into the header area at the top of the page.
  2. Type your name. It will now appear at the top of every page automatically.
  3. Click Go to Footer on the Header & Footer tab (or scroll down and click in the footer area).
  4. Click Page Number → Current Position → Plain Number. A page number appears.
  5. Click Close Header and Footer (the red X on the right) to go back to your normal document.

Quick Styles (Home Tab → Styles Group)

Styles apply a combination of formatting in one click. Built-in styles include Heading 1, Heading 2, Normal, Title. Using heading styles consistently enables automatic Table of Contents generation.

Proofing and Reviewing

Spell Check

Autocorrect

Word Count
Review → Word Count (or shown in Status Bar at bottom)

Working with Tables

Step-by-step: Insert a table with 3 columns and 4 rows
  1. Click in your document where you want the table to appear (the blinking line, called the cursor, shows where it will go).
  2. Click the Insert→Table button. A small grid of squares drops down.
  3. Move your mouse over the grid. As you move, squares light up to show the size. Move until 3 across and 4 down are highlighted (it says "3x4 Table" at the top).
  4. Click once. Word inserts an empty table with 3 columns and 4 rows.
  5. Click inside any block (called a cell ) and type. Press the Tab key to jump to the next cell. (Pressing Tab in the last cell makes a new row.)

Two new tabs appear whenever your cursor is inside a table: Table Design (colours, borders, styles) and Table Layout (add/delete rows, merge cells, alignment).

Objects and Illustrations (Insert Tab)

  • Pictures — insert from file or online; resize using corner handles
  • Shapes — draw rectangles, circles, arrows; format with fill colour and borders
  • WordArt — decorative text effects
  • SmartArt — visual diagrams (org charts, lists, cycle diagrams)
  • Screenshots — captures an image of your screen or a window
  • Text Box — a box you can place anywhere on the page to type in
TEXT WRAPPING

When inserting pictures or shapes, change the text wrapping to control how text flows around the object. Common options: In Line with Text (moves with text), Square or Tight (text wraps around the object), Behind Text / In Front of Text.

Templates

A template is a ready-made document with the layout, fonts and placeholder text already done for you — like a pre-printed form you just fill in, instead of designing a letterhead from scratch every time you write a letter.

Step-by-step: Start a document from a template
  1. Click File → New.
  2. Browse the available templates (Letter, Fax, Report, etc.) or search online templates using the search box.
  3. Click a template to preview it, then click Create.
  4. Replace the placeholder text (often in square brackets, e.g. [Company Name] ) with your own details.

Watermark and Page Colour

A watermark is faint text or an image (like "DRAFT" or "CONFIDENTIAL") shown behind the normal text on every page — visible enough to notice, faint enough not to interfere with reading.

TaskHow
Insert a text watermarkDesign→Watermark → choose a preset (e.g. CONFIDENTIAL) or Custom Watermark… to type your own text
Remove a watermarkDesign → Watermark → Remove Watermark
Change the page background colourDesign → Page Color → choose a colour (this colour shows on screen but does not print by default)

Hyphenation

Hyphenation breaks long words across the end of a line with a hyphen, instead of pushing the whole word onto the next line — this keeps the right-hand edge of justified text looking neater.

  • Turn on automatic hyphenation: Layout → Hyphenation → Automatic
  • Turn off hyphenation: Layout → Hyphenation → None

Protecting a Document

Sometimes you want to share a document but stop people from changing it by accident — like handing someone a form where only certain blanks can be filled in.

Step-by-step: Restrict editing
  1. Click Review → Restrict Editing.
  2. Under "Editing restrictions", tick the checkbox and choose a restriction level (e.g. "No changes (Read only)").
  3. Click Yes, Start Enforcing Protection, then optionally type a password so only you can remove the protection later.
PASSWORD-PROTECTING THE FILE ITSELF

Restrict Editing protects the content while the file is open. To stop the file from opening at all without a password, use File → Info → Protect Document → Encrypt with Password instead.

View Options

Print Layout

Simplified view without page margins — good for fast editing

Side-by-side

EXAM TIP — Common Word Processing Tasks

In a practical exam, marks are awarded for precise formatting. Pay attention to: exact font sizes and types, correct alignment, proper header/footer content, correct use of styles, and whether you save in the correct format (.docx). Always save frequently!
