CAT Glossary & Acronyms All Grades

Every key CAT term in one place, with plain definitions. Type in the box to filter instantly — great for the matching and acronym questions that open every Paper 2.

A

Absolute cell reference
A spreadsheet reference locked with $ (e.g. $B$2) that does not change when the formula is copied.
Accessibility
How usable a device or program is for people with disabilities (e.g. screen readers, voice input, large-key keyboards).
Activation code / product key
A code entered when installing software to prove a legal licence and limit how many devices may install it.
Adware
Malware that displays unwanted adverts and pop-up windows.
Analogue
A continuously varying signal, unlike digital which uses discrete 0s and 1s.
Antivirus
A utility that scans for, quarantines and removes malware; must be kept updated.
Attribute (file)
A setting that changes how a file behaves, e.g. Read-only or Hidden.
Attribute (HTML)
Extra information added inside an HTML tag, e.g. href or src.
AUP (Acceptable Use Policy)
The rules users must agree to when using an organisation's network.

B

Backup
A second copy of data kept safely so it can be recovered if the original is lost, stolen or damaged.
Bandwidth
The amount of data that can be transferred over a connection per second (measured in Mbps).
Biometrics
Security based on a physical trait — fingerprint, face or iris.
Blockchain
A database that keeps an ordered ledger of records, such as cryptocurrency transactions.
Blog
A website updated regularly with dated posts or articles.
Bluetooth
Short-range wireless used to connect nearby devices — an example of a PAN.
Bookmark / favourite
A saved link so you can return to a web page quickly.
Broadband
A high-speed, always-on internet connection.
BYOD (Bring Your Own Device)
A policy allowing people to use their personal devices on an organisation's network.
Byte
A unit of storage (8 bits). Sizes go KB → MB → GB → TB (×1024 each).

C

Cache
Stored copies of recently used web data so pages load faster on the next visit.
CAP (data cap)
The maximum data allowed per month before extra charges or throttling.
CAPTCHA
A test that tells a real human apart from an automated bot (e.g. distorted text or image puzzles).
Cell
The box where a row and column meet in a spreadsheet, identified by e.g. B3.
Cloud computing
Using software and processing over the internet on remote servers instead of your own PC.
Cloud storage
Storing data on remote internet servers (OneDrive, Google Drive, Dropbox).
Compression (zip)
Reducing a file's size and/or bundling several files into one archive.
Conditional formatting
Automatically changing a cell's appearance based on its value.
Convergence
Combining several technologies into one device (a smartphone is the classic example).
Cookie
A small file a website stores on your device to remember you or your settings.
Copyright
Legal protection of original work; using it without permission is infringement.
CPU (Central Processing Unit)
The "brain" that processes instructions; speed measured in GHz.
Cryptocurrency
A digital, hard-to-trace currency not controlled by banks (e.g. Bitcoin).
Cyberbullying
Using digital platforms to threaten, humiliate or harass someone.

D

Data
Raw, unprocessed facts that have no meaning on their own.
Database
An organised collection of related data stored in tables (e.g. Access).
DDoS (Distributed Denial of Service)
Flooding a server with traffic from many computers so it can't respond.
Defragmentation
Reorganising scattered file pieces on a disk so files open faster — it does not free space.
Digital divide
The gap between people who have access to ICT and those who do not.
Digital footprint
The trail of data you leave behind through your online activity.
Driver
Software that lets the operating system communicate with and control a hardware device.
Drop Cap
A large decorative first letter of a paragraph, spanning two or more lines.

E

Encryption
Scrambling data so that only authorised people can read it.
Ergonomics
Designing a workspace and equipment to fit the user safely and prevent injury.
EULA (End User Licence Agreement)
The terms and conditions you accept when installing or using software.
E-waste
Discarded electronic devices; they contain toxic materials and must be recycled responsibly.
Extranet
An intranet extended to give limited access to selected outsiders (suppliers, clients).

F

Fibre
A fast, stable cabled internet connection that sends light through glass strands.
Field
A single item of data in a database record (a column), e.g. Surname.
Firewall
Hardware or software that blocks unauthorised access to or from a network.
Flash drive
Small, portable USB storage using flash memory.
Fragmentation
When the pieces of files become scattered across a disk, slowing access.
Freeware
Software that is free to use but whose source code is not shared.

G

Geo-tagging
Adding the GPS coordinates of where a photo was taken into the file's metadata.
GPS (Global Positioning System)
Uses satellites to work out a device's exact location.
GPU (Graphics Processing Unit)
Hardware that renders graphics; a dedicated GPU is needed for editing, 3D and gaming.
Green computing
Using computers in an environmentally responsible way (save energy, recycle e-waste, print less).
GUI (Graphical User Interface)
Interacting with a computer through windows, icons, menus and a pointer.

H

HAN (Home Area Network)
A network of all the devices in a home.
Hardware
The physical parts of a computer that you can touch.
HDD (Hard Disk Drive)
Magnetic spinning-disk storage; cheaper per GB but slower and less durable than an SSD.
HDMI
A port and cable that carries high-definition video and audio together.
Hotspot
A Wi-Fi access point that shares an internet connection with nearby devices.
HTML (HyperText Markup Language)
The markup language used to build and structure web pages.
HTTP / HTTPS
Web protocols; HTTPS is the secure, encrypted version (look for the padlock).
Hyperlink
Clickable text or an image that jumps to another page, file or place in a document.

I

ICT (Information & Communication Technology)
Technology used to process, store and share information.
Information
Processed data that has meaning and can support a decision.
Information overload
Being overwhelmed by too much information to process or make decisions.
Input device
Hardware used to enter data or instructions (keyboard, mouse, scanner, microphone).
Input mask
A database field setting that controls the format in which data is entered.
Intranet
An organisation's private internal network, like a private version of the internet.
IoT (Internet of Things)
Everyday devices fitted with sensors and internet connectivity that exchange data.
IP address
A unique number that identifies a device on a network.
ISP (Internet Service Provider)
A company that provides (sells) internet access, e.g. Afrihost, MWEB.

K – L

Keylogger
Malware that secretly records every key you press to steal passwords.
LAN (Local Area Network)
A network within one building or site.
LTE / 4G / 5G
Mobile cellular data technologies; 5G is the fastest with the lowest latency.

M

Malware
Any malicious software designed to damage a system or gain unauthorised access.
Metadata
Data about a file (author, date, size, type) — not its actual content.
Modem
Connects a network to the internet through the ISP.
Motherboard
The main circuit board that connects all the components of a computer.
Multitasking
The operating system running several programs seemingly at the same time.

N

Netiquette
Good manners and correct behaviour when communicating online.
NFC (Near Field Communication)
Very short-range wireless for tap-to-pay and similar.
NIC (Network Interface Card)
Hardware (a network adapter) that connects a device to a network.

O

OCR (Optical Character Recognition)
Software that converts scanned printed text into editable text.
OLE Object
An Access data type that embeds an object from another program (image, document) in a field.
Open-source
Software whose source code is freely available to view, change and share.
Operating system (OS)
System software that runs and manages the computer and its hardware (Windows, Android).
OTP (One-Time Password)
A single-use code (often via SMS) for one login, adding a second layer of security.
Output device
Hardware that presents processed data to the user (monitor, printer, speaker).

P

PAN (Personal Area Network)
A network of devices around one person, e.g. a phone paired with earbuds (Bluetooth).
Patch
A small software update that fixes a bug or closes a security vulnerability.
PAT (Practical Assessment Task)
A year-long research project that counts towards the CAT mark.
Peripheral
A hardware device connected to but outside the main system unit (printer, mouse).
Pharming
Redirecting a user to a fake website (even with the correct address) to steal details.
Phishing
A fake message pretending to be from a trusted source to trick you into revealing information.
Piracy
Illegally copying, sharing or using software.
Plagiarism
Presenting someone else's work (including AI output) as your own.
Plug-and-play
Hardware that configures itself automatically as soon as it is connected.
POPIA
South Africa's law for protecting people's personal information.
Primary key
A database field that is unique for every record (no duplicates or blanks).
Proprietary software
Commercial software you must buy and licence; source code is not shared.
Protocol
A set of rules for communication between devices (e.g. HTTP, TCP/IP).

Q – R

QR code
A 2D barcode that links to a URL or data when scanned by a camera.
RAM (Random Access Memory)
Temporary, volatile working memory that holds data in use; its contents are lost when the power is off.
Ransomware
Malware that encrypts your files and demands a ransom to unlock them.
Record
All the data about one item in a database (a row in a table).
Relative cell reference
A spreadsheet reference (e.g. B2) that adjusts automatically when the formula is copied.
Resolution
The number of pixels on a screen or in an image; higher resolution = sharper.
RFID (Radio Frequency Identification)
A tag read wirelessly, used for access cards, stock and anti-theft.
ROM (Read Only Memory)
Permanent, non-volatile memory that holds the start-up (boot) instructions.
Router
A device that directs data between networks and shares an internet connection.
RSI (Repetitive Strain Injury)
Injury to the wrists/forearms from repeated movements and poor posture.

S

SaaS (Software as a Service)
Software used over the internet on a subscription, rather than installed on your PC.
Search engine
A tool that finds web pages by keyword (e.g. Google, Bing).
Section break
A Word divider that lets different pages of a document be formatted differently.
Server
A powerful computer that provides services and resources to client computers on a network.
Service pack
A collection of updates and patches bundled together into one large update.
Social engineering
Tricking people into giving up confidential information rather than hacking a system.
Software
The programs and instructions that tell hardware what to do.
SOHO (Small Office / Home Office)
A small-business or home-based computing setup.
Spyware
Malware that secretly monitors your activity and sends it to an attacker.
SSD (Solid State Drive)
Fast, durable flash storage with no moving parts; more expensive per GB than an HDD.
Streaming
Playing media in real time over the internet without saving a full copy.
Switch
Connects devices within a LAN and forwards data only to the intended device.
System software
Software that runs and manages the computer (the operating system and utilities).

T – U

Throttling
An ISP deliberately slowing your connection after a data cap or during peak times.
Traffic shaping
An ISP prioritising some kinds of traffic and slowing others to manage congestion.
Trojan
Malware disguised as useful software that takes over once installed.
UPS (Uninterruptible Power Supply)
A battery backup that keeps a computer running briefly during a power failure.
URL (Uniform Resource Locator)
The full web address of a page.
USB (Universal Serial Bus)
A common port and connector for peripherals and storage.
Utility
System software that maintains the computer (antivirus, disk cleanup, defragmenter).

V

Validation
Checking that entered data is reasonable and within an allowed range or format.
Verification
Checking that data was captured correctly, e.g. re-typing a password to confirm it.
Virus
Malware that attaches to a file and spreads when that file is opened or run.
VLOOKUP
An Excel function that looks up a value in the first column of a table and returns a value from the same row.
VoIP (Voice over Internet Protocol)
Making voice calls over the internet (e.g. a WhatsApp call).
Volatile
Memory that loses its contents when the power is switched off (RAM).
VPN (Virtual Private Network)
A secure, encrypted connection giving safe remote access to a private network.
VR / AR
Virtual Reality (a fully simulated environment) / Augmented Reality (digital elements overlaid on the real world).

W

WAN (Wide Area Network)
A network spanning a large area; the internet is the largest WAN.
Watermark
Faint text or an image placed behind a document's content (e.g. "DRAFT").
Wi-Fi
Wireless networking using radio waves — a wireless LAN (WLAN).
WLAN (Wireless LAN)
A local area network that connects devices wirelessly.
Worm
Malware that copies itself and spreads across networks without needing a host file.
WWW (World Wide Web)
The collection of websites and web pages accessed over the internet.
WYSIWYG
"What You See Is What You Get" — the screen shows the document exactly as it will print or publish.
