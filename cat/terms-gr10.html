Grade 10 — Year Planner Grade 10

Based on the 2026 CAT Annual Teaching Plan. Each term shows the practical (Solution Development) and theory (Systems/Network/Internet Technologies, Information Management, Social Implications) topics covered.

A year planner — not a test scope

This shows when each topic is taught during the year — it is not a list of what a particular test covers. Any work done so far this year, plus the foundational work from earlier grades, can still be assessed, so don't study only the current term's topics.

1
Term 1 — Foundations & Word Processing
Assessment: SBA Task 1 — Theory Test (min 50 marks)
Practical (Solution Development)
  • Word Processing — file mgmt, text, formatting, paragraphs, page layout, objects
Theory
  • Concepts of Computing
  • Hardware (input, output, storage)
  • Software & Licensing
  • Social Implications (ergonomics, green)
2
Term 2 — Spreadsheets & Mid-Year Exam
Assessment: SBA Task 2 — Practical Test + SBA Task 3 — June Exam (P1 + P2)
Practical (Solution Development)
  • Word Processing — tables, bullets, tabs, headers/footers, cover page
  • Spreadsheets — intro, cells, basic functions, error indicators
Theory
  • Hardware (extended — scanning, biometric, processing)
  • Software (drivers, utilities, licensing)
  • Networks (intro — LAN, PAN, HAN)
  • Information Management (intro to PAT)
3
Term 3 — HTML, Internet & PAT Phase 1
Assessment: SBA Task 4 — Practical Test + SBA Task 5 — Theory/Alternative. PAT Phase 1 completed.
Practical (Solution Development)
  • Spreadsheets — TODAY, COUNTIF, sorting, sheets
  • HTML & Web Design — tags, formatting, font, structure
  • Word Processing — styles, TOC, columns, watermark
  • Presentations (PAT, plus ~15% of P1)
Theory
  • Networks (PAN/HAN, devices, ISP, Internet access)
  • Internet & WWW, E-Communication
  • Social (digital citizenship, POPIA, OTP)
4
Term 4 — Charts, PAT Phase 2 & Final Exam
Assessment: SBA Task 6 — End-of-Year Exam (P1 120 + P2 120). PAT Phase 2 completed (100 marks).
Practical (Solution Development)
  • Spreadsheets — charts/graphs, integration, troubleshooting
  • Word — help, integration, working with documents
Theory
  • Social (impact on society, computer crime)
  • Content using case studies (all topics — revision)
GRADE 10 EXAM STRUCTURE

Paper 1 (Practical, 120 marks, 2.5 hrs): Word Processing (~40%), Spreadsheets (~30%), Presentations (~15%), General/Integration (~15%).
Paper 2 (Theory, 120 marks, 2.5 hrs): Short questions, Systems Technologies, Internet/Network, Information Management, Social Implications, Solution Development.
