Spreadsheets Grade 12

Grade 12 adds complex functions (Nested IF, AND/OR, VLOOKUP/HLOOKUP, COUNTIFS/SUMIFS, text functions, date/time functions) and advanced charts including stacked graphs and trendlines.

Complex Functions

Nested IF

A nested IF places one IF inside another to test multiple conditions:

Nested IF — Grade classification
=IF(B2>=80, "Distinction",
  IF(B2>=60, "Merit",
    IF(B2>=50, "Pass",
      "Fail")))
COUNTING BRACKETS

Each IF needs a closing bracket. With 3 nested IFs, you need 3 closing brackets ))) at the end. Count: one ) for each IF opened.

AND / OR Functions

AND and OR
<-- AND: all conditions must be TRUE -->
=IF(AND(B2>=50, C2>=50), "Pass", "Fail")

<-- OR: at least one condition must be TRUE -->
=IF(OR(B2>=80, C2>=80), "Has Distinction", "No Distinction")

<-- Combined AND + OR -->
=IF(AND(A2="Active", OR(B2>100, C2>100)), "Yes", "No")

LOOKUP Functions

VLOOKUP — looks up values in a vertical table
=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

Example: Look up student ID in B2, find their mark in column 3 of the lookup table
=VLOOKUP(B2, $E$2:$G$20, 3, FALSE)

Arguments:
  lookup_value = the value to search for (B2)
  table_array  = the range to search in ($E$2:$G$20) — always ABSOLUTE!
  col_index    = which column in the table to return (3 = 3rd column)
  range_lookup = FALSE = exact match; TRUE = approximate match

Error: if not found, returns #N/A
HLOOKUP — looks up values in a horizontal table
=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])

Same as VLOOKUP but searches across rows instead of down columns
Step-by-step: Build a VLOOKUP to find a learner's mark
  1. Click the empty cell where you want the answer to appear.
  2. Type =VLOOKUP( to start the function.
  3. Lookup value: click the cell that holds the thing you are searching for (e.g. the learner ID in B2), then type a comma ,
  4. Table array: highlight the whole lookup table (all columns), then press F4 once. This adds dollar signs (e.g. $E$2:$G$20) so the table will not shift when you copy the formula. Type a comma ,
  5. Column number: count which column of that table holds the answer, counting from the LEFT starting at 1. Type that number (e.g. 3 for the 3rd column), then a comma ,
  6. Exact match: type FALSE (this means "only give me an exact match").
  7. Type the closing bracket ) and press Enter. Final formula: =VLOOKUP(B2,$E$2:$G$20,3,FALSE)
EXAM TIP — VLOOKUP

Most common VLOOKUP mistakes: (1) not using absolute reference for table_array ($E$2:$G$20) — it shifts when copied; (2) wrong col_index number — count from the LEFT of the table_array, not from column A; (3) using TRUE instead of FALSE for exact match — always use FALSE in CAT unless told otherwise.

COUNTIFS and SUMIFS

These extend COUNTIF/SUMIF to support multiple criteria:

COUNTIFS and SUMIFS
COUNTIFS: count cells meeting MULTIPLE conditions
=COUNTIFS(A2:A20, "Male", B2:B20, ">=50")
<-- counts rows where A = "Male" AND B >= 50

SUMIFS: sum values meeting MULTIPLE conditions
=SUMIFS(sum_range, criteria_range1, criteria1, criteria_range2, criteria2)
=SUMIFS(C2:C20, A2:A20, "Cape Town", B2:B20, ">100")
<-- sums C where A = "Cape Town" AND B > 100

ROUNDUP and ROUNDDOWN

ROUNDUP / ROUNDDOWN
=ROUNDUP(A1, 0)    <-- always rounds UP to nearest integer (3.1 → 4)
=ROUNDDOWN(A1, 0) <-- always rounds DOWN to nearest integer (3.9 → 3)
=ROUND(A1, 0)      <-- rounds to nearest (3.5 → 4; 3.4 → 3)

Date and Time Functions

FunctionReturnsExample

MONTH(date)Month number (1–12)=MONTH(A1) → 6

Age calculation example
=YEAR(TODAY()) - YEAR(B2)   <-- approximate age in years
=DAYS(TODAY(), B2) / 365.25  <-- more precise age

Subtotal Feature

Subtotal creates grouped summaries within filtered data (unlike SUM which includes hidden rows):

  • Data → Subtotal → choose column to group by, function (SUM/AVG/COUNT) and column to calculate
  • Creates outline levels — click 1, 2, 3 buttons to expand/collapse groups
  • Remove: Data → Subtotal → Remove All

EXAM TIP — Function Identification

A common Grade 12 exam task: given a scenario, choose the CORRECT function. Key decision points:
• Need to find a value in a table? → VLOOKUP
• Multiple conditions to count? → COUNTIFS
• Multiple conditions to add? → SUMIFS
• Grade classification? → Nested IF
• Extract part of a code? → LEFT / RIGHT / MID
• Calculate how many days since a date? → DAYS or TODAY()
