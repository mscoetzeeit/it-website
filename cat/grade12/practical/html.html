HTML & Web Design Grade 12

Grade 12 HTML focuses on advanced table formatting (border, cellpadding, cellspacing, alignment, merging), designing complete web pages for specific scenarios, and applying good web design principles.

GRADE 12 HTML — ALL PREVIOUS TAGS STILL APPLY

All Grade 10 (basic structure, text tags, font tag, hr, br, body bgcolor) and Grade 11 (lists, images, tables, links, comments) tags are still examinable. Grade 12 deepens table formatting knowledge and web design principles.

Advanced Table Formatting

Table Attributes — Complete Reference

AttributeApplied toValues / ExampleEffect
border<table>border="1"Border around table and cells (0 = no border)
cellpadding<table>cellpadding="5"Space (px) INSIDE cells between content and cell wall
cellspacing<table>cellspacing="3"Space (px) BETWEEN cells
width<table>, <td>width="80%" or width="400"Width of table or cell (px or %)
height<table>, <td>height="50"Height of table or cell in pixels
bgcolor<table>, <tr>, <td>bgcolor="lightblue"Background colour
align<table>, <td>, <th>left | center | rightHorizontal alignment of table on page, or content in cell
valign<td>, <th>top | middle | bottomVertical alignment of content within cell
colspan<td>, <th>colspan="2"Merges cell across 2 (or more) columns
rowspan<td>, <th>rowspan="3"Merges cell across 3 (or more) rows

Merging Cells — colspan and rowspan Together

Complex table with merged cells
<table border="1" cellpadding="5">
  <tr>
    <th rowspan="2">Name</th>      <!-- spans 2 rows vertically -->
    <th colspan="2">Scores</th>  <!-- spans 2 columns horizontally -->
  </tr>
  <tr>
    <th>Term 1</th>                  <!-- 2nd row of "Scores" header -->
    <th>Term 2</th>
  </tr>
  <tr>
    <td>Sipho</td>
    <td>72</td>
    <td>85</td>
  </tr>
</table>

Good Web Page Design Principles

Grade 12 expects you to evaluate and apply these principles:

PrincipleGood practiceBad practice
ReadabilityHigh contrast (dark text on light bg), min 12pt body text, clear headingsLight grey text on white, tiny font, no spacing
NavigationConsistent menu, clear labels, max 3 clicks to any pageNo back button, broken links, confusing structure
ConsistencySame colours, fonts and header on all pagesDifferent layout on every page
LayoutGrid-based, balanced text and images, breathing roomCluttered, inconsistent margins, too much text
ColourMax 3 main colours, complementary palette, accessible contrastClashing colours, too many colours, red text on green
TypographyMax 2 font families, heading clearly larger than body5 different fonts, all the same size
Loading speedCompress images, avoid large filesHuge uncompressed images, embedded videos on every page

</body>
</html>
EXAM TIP — HTML Tag Sheet

In a practical exam, an HTML tag reference sheet is provided. Read it carefully — the exam uses only the tags listed. Most common Grade 12 mistakes: (1) confusing cellpadding (inside cells) vs cellspacing (between cells); (2) counting columns wrong when using colspan/rowspan — draw the table on paper first to plan merges; (3) forgetting to close tags like </table> or </tr>.
