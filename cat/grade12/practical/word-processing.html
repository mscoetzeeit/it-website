Word Processing Grade 12

Grade 12 reinforces all Grade 10 and 11 Word skills and adds: advanced mail merge (multiple data sources), tracking changes, bookmarks, cross-references, advanced linking/embedding and professional document production.

GRADE 12 APPROACH

Grade 12 exams include ALL Grade 10 and 11 Word skills, so revise those pages too. New Grade 12 content focuses on problem-solving and integration — applying skills in unfamiliar scenarios.

Advanced Mail Merge — Multiple Data Sources

In Grade 12, mail merge data sources extend beyond Word tables and Excel:

Data SourceNotes
Word tableTable in a .docx file (first row = field names)
Excel spreadsheetFirst row = field names; sheet name selected during merge
Access databaseChoose a table or query as the data source
CSV fileComma-separated values; compatible with most applications
Email listUse Mailings → Finish → Send Email Messages to merge directly to emails
Step-by-step: Create personalised letters with Mail Merge
  1. Open (or type) your letter in Word — leave gaps where the personal details (name, address) must go.
  2. Click the Mailings→Start Mail Merge button, then choose Letters.
  3. Click Select Recipients → Use an Existing List… and browse to your data source (an Excel file, Access database or Word table). Click Open.
  4. Click in the letter where a name should go. Click Insert Merge Field and choose the field (e.g. FirstName). A placeholder like «FirstName» appears.
  5. Repeat step 4 for every piece of personal information (surname, address, etc.).
  6. Click Preview Results to check how a real letter looks. Use the arrows to flip through each recipient.
  7. Click Finish & Merge → Edit Individual Documents → All → OK. Word creates one new document containing a personalised letter for every recipient.

Mail Merge for Labels

  1. Mailings → Start Mail Merge → Labels
  2. Select label type (Avery, etc.) and size
  3. Select Recipients → choose data source
  4. Insert Address Block or individual merge fields
  5. Update Labels (applies format to all labels)
  6. Finish & Merge → Print or Edit

Links, Bookmarks and Cross-References

Bookmarks

A bookmark marks a specific location in a document so you can navigate to it or link to it.

  • Insert → Bookmark → type name → Add
  • Navigate: Ctrl+G → Bookmark → select name → Go To
  • Create hyperlink to bookmark: Ctrl+K → This Document → select bookmark

Hyperlinks

  • Link to a website: Ctrl+K → Existing File or Web Page → type URL
  • Link to another file: Ctrl+K → Existing File or Web Page → browse to file
  • Link to email address: Ctrl+K → Email Address
  • Remove hyperlink: right-click → Remove Hyperlink

Cross-References

A cross-reference creates a link to another part of the same document (e.g. "See Figure 3 on page 5").

  • References → Cross-reference → choose Reference type (Heading, Figure, Table, Bookmark) and Insert reference to (Page number, Heading text, etc.)
  • When the document changes, select all (Ctrl+A) then press F9 to update all cross-references

Tracking Changes

Tracking changes records all edits made to a document so reviewers can see what was changed.

  • Enable tracking: Review → Track Changes (toggle on/off) — Ctrl+Shift+E
  • Changes appear in a different colour with strikethrough for deletions and underline for insertions
  • Show Markup: Review → Show Markup — choose what types to show
  • Accept change: right-click change → Accept Change (keeps the edit)
  • Reject change: right-click → Reject Change (reverts to original)
  • Accept/Reject All: Review → Accept All Changes / Reject All Changes

Reviewing Tools

Comments
Review → New Comment — adds an annotation balloon without changing the text. Useful for feedback.
Spelling & Grammar
F7 or Review → Spelling & Grammar — checks the entire document
Word Count
Review → Word Count — shows words, characters, paragraphs

Line Breaks and Pagination

Widow and Orphan Control

Widow
Last line of a paragraph appearing alone at the top of a new page (looks bad)
Orphan
First line of a paragraph appearing alone at the bottom of a page (looks bad)

Control these: select paragraph → Home → Paragraph dialog → Line and Page Breaks tab → check "Widow/Orphan control"

Other Line/Page Break Options

  • Keep with next: keeps a heading on the same page as the paragraph that follows
  • Keep lines together: prevents a paragraph from being split across pages
  • Page break before: forces paragraph to start on a new page

Professional Document Production

Grade 12 expects complete, professional documents that combine all skills:

  • Consistent styles (Heading 1, Heading 2, Normal)
  • Automatic Table of Contents
  • Headers/footers with fields (page number, date, author)
  • Properly formatted bibliography using References → Citations
  • Images with captions and cross-references
  • Section breaks for different page orientations/column layouts
  • Mail merge for personalised letters or labels
EXAM TIP — Tracking Changes

In the practical exam you may be asked to: accept/reject specific changes, add a comment, or turn tracking on/off. Know the difference between Accept (keeps the edit) and Reject (removes the edit and restores the original). Changes appear in red by default; each reviewer gets a different colour.
