HTML & Web Design Grade 10

HTML (HyperText Markup Language) is the language used to create web pages. In CAT, HTML is coded in a text editor like Notepad++ and viewed in a browser.

What is HTML?

HTML = HyperText Markup Language. It uses tags (instructions wrapped in angle brackets) to structure and format content on a web page. Browsers read the HTML and display it visually.

HTML EDITOR

In CAT, use Notepad++ to write HTML (it highlights your code in colour). Save the file with a .html extension. Open it in a browser (Chrome, Edge) to see the result. You do NOT need an internet connection to view local HTML files.

Step-by-step: Create your first HTML file in Notepad++
  1. Open Notepad++ (click the Start button, type "Notepad++" and click it). A blank white page appears.
  2. Type your HTML code into the page (use the examples below).
  3. Click File→Save As at the top.
  4. In the File name box, type a name ending in .html — for example mypage.html (the .html part is very important, it tells the computer this is a web page).
  5. Click Save.
  6. Find your saved file in its folder and double-click it. It opens in a web browser (Chrome/Edge) so you can see the result.
  7. To make changes: edit the code in Notepad++, save again (Ctrl + S), then go back to the browser and press F5 to refresh.

Tag Syntax

Most HTML tags come in pairs: an opening tag and a closing tag. The closing tag has a forward slash / before the tag name.

Syntax
<tagname>Content goes here</tagname>

Some tags are self-closing — they have no content and no closing tag: <br> and <hr>

Basic HTML Page Structure

Basic HTML structure
<html>
  <head>
    <title>Page Title Here</title>
  </head>
  <body>
    <!-- All visible content goes here -->
    <h1>My First Web Page</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Grade 10 HTML Tag Reference

TagClosing TagPurpose
<html></html>Marks the start and end of the HTML document
<head></head>Contains metadata (page info not shown on page)
<title></title>Sets the browser tab/window title
<body></body>Contains all visible page content
<h1> to <h6></h1> etc.Headings — h1 is largest, h6 is smallest

<b></b>Bold text
<i></i>Italic text
<u></u>Underlined text

Attributes

Attributes add extra information to a tag. They go inside the opening tag as name="value" pairs.

Attribute syntax
<tagname attribute="value">content</tagname>

The Font Tag

The <font> tag controls the appearance of text using attributes:

Font tag examples
<font face="Arial" size="4" color="red">
  This text is Arial, size 4, red colour
</font>
AttributeValuesEffect
faceFont name (e.g. "Arial", "Times New Roman")Sets the typeface
size1 (smallest) to 7 (largest)Sets the text size
colorColour name ("red") or hex (#FF0000)Sets text colour

Horizontal Rule Attributes

hr with attributes
<hr width="50%" size="3">   <!-- 50% wide, 3px thick -->

Body Background Colour

Setting page background colour
<body bgcolor="lightblue">
  <!-- All page content -->
</body>

Colour values can be: colour names ("red", "yellow", "lightgreen") or hex codes ("#FF0000").

Complete Example — Grade 10 HTML Page

Full example
<html>
<head>
  <title>Ms Coetzee CAT</title>
</head>
<body bgcolor="lightyellow">

  <h1><font color="darkblue">Welcome to Grade 10 CAT</font></h1>
  <hr width="80%">

  <h2>About this subject</h2>
  <p>CAT teaches us to use <b>application software</b> effectively.</p>
  <p>We study <i>Word, Excel, Access</i> and <u>HTML</u>.</p>

  <h3>Practical Tools:</h3>
  <p><font face="Arial" size="3" color="green">Microsoft Word</font></p>
  <p><font face="Arial" size="3" color="green">Microsoft Excel</font></p>

</body>
</html>
EXAM TIP — Common HTML Mistakes

1. Forgetting to close tags (e.g. <b> without </b>) — everything after becomes bold.
2. Wrong quotes — use straight quotes " not curly quotes ".
3. Spelling colour as "color" — HTML uses American spelling: color, not colour.
4. <hr> and <br> do NOT need closing tags.
5. The <title> only changes the browser tab text — not what you see on the page.
