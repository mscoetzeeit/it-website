Networks Grade 10

A network connects computers and devices so they can communicate and share resources. This topic covers types of networks, devices, communication media and how to obtain Internet access.

What is a Network?

A computer network is two or more computing devices connected together to share resources and communicate. Networks can be wired (using cables) or wireless.

Aims and Objectives of Networks

  • Communication — send emails, messages, video calls
  • Share hardware — multiple computers share one printer or scanner
  • Share software — run programs stored on a central server
  • Share data/information — access files stored on a shared drive
  • Security — centralised access control; who can log in and what they can see

PAN vs HAN

PAN: personal devices around ONE person (e.g. phone + smartwatch + earbuds). Range is typically under 10 m.
HAN: all devices in a home (phones, laptops, smart TV, smart fridge). Usually uses a home Wi-Fi router. Larger than a PAN.

Network Devices

Modem
Modulator-Demodulator — converts digital signals to analogue and back; connects the local network to the Internet via an ISP
Router
Directs data packets between different networks; connects your home/school network to the Internet; often combined with modem
Switch
Connects devices within a LAN; forwards data only to the intended device (more efficient than a hub)

Communication Media (How devices connect)

Wired connections

  • UTP cable (Ethernet) — twisted copper wire; used in LANs; reliable and fast
  • Fibre optic — uses light pulses through glass/plastic; very fast; expensive; used for long distances and ISP backbones

Wireless connections

Wi-Fi
Wireless connection using radio waves; used in homes, schools, public hotspots; range typically 30–100 m indoors
Bluetooth
Short-range wireless (up to ~10 m); used to pair devices (phone to earbuds, keyboard, speaker)
NFC
Near Field Communication; very short range (~4 cm); used for tap-to-pay, access cards

Pairing Devices

Pairing is the process of connecting two Bluetooth devices so they can communicate. Steps: enable Bluetooth on both devices → make one device discoverable → select it on the other device → confirm the pairing code.

Obtaining Internet Access

To connect to the Internet you need:

  • A computing device (phone, laptop, tablet)
  • A modem/router OR mobile data (4G/5G)
  • An account with an ISP (Internet Service Provider)
ISP — Internet Service Provider

An ISP is a company that provides Internet access to homes and businesses. Examples in SA: Telkom, Vodacom, MTN, Afrihost, Rain. They provide a connection type (ADSL, fibre, LTE, 5G) and you pay a monthly fee for data or uncapped access.

Basic Network Security

  • Password protection — only authorised users can log in
  • Firewall — software/hardware that monitors and filters network traffic
  • Encryption — scrambles data so it cannot be read if intercepted
  • Access rights — users can only access files/folders they are permitted to see
EXAM TIP — Advantages vs Disadvantages of Networks

Advantages: resource sharing, communication, centralised data storage, easy software updates.
Disadvantages: expensive to set up, security risks (hacking, viruses spread across network), dependent on server (if server fails, everyone is affected), requires technical support.
