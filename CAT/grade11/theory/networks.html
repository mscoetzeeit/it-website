Networks (LAN/WLAN) Grade 11

Grade 11 focuses on Local Area Networks (LAN and WLAN) — their components, connection types, speed, security and acceptable use policies. Builds on Grade 10 network basics.

LAN vs WLAN

LAN (Local Area Network)WLAN (Wireless Local Area Network)
ConnectionWired (Ethernet/UTP cables)Wireless (Wi-Fi radio waves)
Speed100 Mbps – 10 Gbps (very fast)Generally slower than wired; varies by distance
ReliabilityVery reliable — no interferenceCan suffer interference from walls, other devices
SetupRequires cables — harder to installEasy — just connect to Wi-Fi network
SecurityMore secure — harder to interceptLess secure — signals can be intercepted if not encrypted
MobilityLimited — device must be near a portMove anywhere in the coverage area

Basic Network Components

Workstation
A computer on the network that users work on (client in client-server; peer in peer-to-peer)
Server
A powerful computer that provides services to other computers (file storage, printing, email, authentication)
NIC (Network Interface Card)
Hardware that connects a device to a network — wired (Ethernet port) or wireless (Wi-Fi adapter); every networked device has one
Modem
Converts digital signals to analogue (and back) for transmission over telephone/fibre lines; connects network to ISP
Switch
Connects multiple devices in a LAN; forwards data only to the intended device (more efficient than a hub)
Router
Connects different networks together; directs data between the LAN and the Internet; manages IP addresses (DHCP)
Access Point (WAP)
Wireless Access Point — extends a wired network by providing a Wi-Fi signal for wireless devices

Client-Server vs Peer-to-Peer

Client-ServerPeer-to-Peer (P2P)
StructureDedicated server serves multiple client computersAll computers equal — any can be server or client
Used inSchools, businesses, large organisationsSmall home networks, file sharing (BitTorrent)
AdvantageCentralised control, security, backupsSimple, cheap — no server needed
DisadvantageExpensive server; if server fails, everyone affectedLess secure, harder to manage many computers

Communication Media

Wired

UTP (Unshielded Twisted Pair)
Most common Ethernet cable; copper wire pairs twisted together; cheap; used in homes and offices
Fibre optic
Transmits data as light pulses through glass/plastic fibres; very fast; long distances; expensive; not affected by electrical interference

Wireless

Wi-Fi
Wireless LAN standard; uses radio waves; range ~30–100 m indoors; requires access point/router
Bluetooth
Short-range (~10 m); used to connect peripherals (keyboard, mouse, headphones, phone)
4G / 5G
Mobile cellular data; 5G is faster and lower latency; accessed via SIM card in phone/router

Data transmission speed is measured in Mbps (megabits per second) or Gbps (gigabits per second).

Network Security

Passwords
Strong passwords protect network access (use mix of upper/lower/numbers/symbols; min 8 characters)
Usernames
Identifies who is logging in; linked to specific access rights
Access rights / Permissions
Controls what each user can see and do on the network (Read, Write, Execute, Full Control, No Access)
Encryption
Scrambles data so it cannot be read if intercepted (WPA2/WPA3 for Wi-Fi; HTTPS for web)
Firewall
Hardware or software that monitors and filters network traffic — blocks unauthorised access

Acceptable Use Policy (AUP)

An Acceptable Use Policy is a set of rules that users must agree to when using a school or organisation's network. It defines:

  • What the network may and may NOT be used for
  • Consequences for misuse (e.g. loss of access, disciplinary action)
  • Privacy expectations (IT staff may monitor usage)
  • Guidelines on downloading, installing software and accessing certain websites
BYOD (Bring Your Own Device)

BYOD allows learners/employees to use their personal devices on the school/office network. Benefits: cheaper for school, familiar devices. Risks: malware from personal devices, privacy/security concerns, distraction. Schools need a clear BYOD policy as part of their AUP.

EXAM TIP

Know at least 2 advantages and 2 disadvantages of wired vs wireless networks. Know the names and functions of all network components. Be able to interpret a network advert — e.g. "100 Mbps Ethernet switch with 8 ports" means it connects up to 8 devices at 100 Mbps each.
